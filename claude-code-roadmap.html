바이브코딩 & 코딩 에이전트

🎵 바이브코딩 (Vibe Coding)

자연어로 원하는 것을 설명하면 AI가 코드를 작성하는 새로운 개발 패러다임. 비개발자도 소프트웨어를 만들 수 있는 시대.

  • "이런 기능을 만들어줘"라고 말하면 AI가 구현
  • 프로그래밍 언어를 몰라도 앱 제작 가능
  • 코드의 품질과 방향은 인간이 결정

🤖 코딩 에이전트 (Coding Agent)

코드베이스를 읽고, 파일을 편집하고, 명령을 실행하며 개발 도구와 통합하는 자율적 AI 시스템.

  • 전체 프로젝트 구조를 이해
  • 여러 파일에 걸쳐 작업 수행
  • 터미널 명령, Git 등 직접 실행

⚡ Claude Code란?

Anthropic이 만든 터미널 네이티브 AI 코딩 어시스턴트. 에이전틱 시스템으로 계획, 구현, 검증을 자율적으로 수행합니다.

  • 터미널, IDE, 데스크톱, 웹, 모바일 지원
  • 200K 토큰 컨텍스트 윈도우
  • MCP로 외부 도구 무한 확장

설치 방법 (2026)

Windows 설치 (권장: Native Install)

자동 업데이트를 지원하는 공식 설치 방법.

PowerShell

irm https://claude.ai/install.ps1 | iex

CMD

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd

* Git for Windows 사전 설치 필요

macOS / Linux 설치

Native Install (권장)

curl -fsSL https://claude.ai/install.sh | bash

Homebrew

brew install --cask claude-code

* Homebrew는 자동 업데이트 미지원. brew upgrade claude-code로 수동 업데이트.

기타 설치 방법

  • WinGet: winget install Anthropic.ClaudeCode
  • VS Code: 확장 마켓에서 "Claude Code" 검색
  • JetBrains: Marketplace에서 Claude Code 플러그인 설치
  • Desktop App: 다운로드 페이지
  • Web: claude.ai/code (설치 불필요)

사용 환경 비교

💻 Terminal CLI

가장 강력한 환경. 파일 편집, 명령 실행, Git 연동, 파이프라인 자동화까지 모든 기능 지원.

🖥️ VS Code / Cursor

에디터 내 인라인 diff, @-mentions, 계획 검토. 익숙한 IDE에서 바로 사용.

🔷 JetBrains

IntelliJ, PyCharm, WebStorm 등. 대화형 diff 보기와 선택 컨텍스트 공유.

🖥️ Desktop App

독립 앱. 시각적 diff 검토, 여러 세션 병렬, 반복 작업 예약, 클라우드 세션.

🌐 Web (claude.ai/code)

설치 없이 브라우저에서. 오래 걸리는 작업 실행 후 나중에 확인 가능.

📱 Mobile (iOS)

Claude iOS 앱에서 웹 세션 시작 → /teleport로 터미널 전환 가능.

📅 6주 마스터 커리큘럼

바이브코더가 매주 따라갈 수 있는 실전 학습 계획. 각 주차를 클릭하면 상세 내용이 펼쳐집니다.

📅 Week 1 — 설치와 첫 경험 난이도: 입문

목표: Claude Code 설치하고 첫 프로젝트에서 대화하기

  • Claude Code 설치 (Native Install 권장)
  • Pro 구독 또는 API 키 설정
  • 첫 프로젝트 폴더에서 claude 실행
  • 간단한 요청: "이 코드를 설명해줘"
  • CLAUDE.md 파일 직접 작성해보기
  • 공식 Quickstart 가이드 완주

과제: 내 프로젝트 폴더에서 CLAUDE.md 작성 후 캡처 공유

📅 Week 2 — 핵심 명령어 & Plan Mode 난이도: 기초

목표: 명령어 체계 마스터 + Plan Mode 활용법 습득

  • /help, /clear, /compact, /context 실습
  • Shift+Tab으로 Plan Mode 전환 연습
  • Plan Mode에서 기능 설계 → Act Mode로 구현
  • @ 멘션으로 파일/폴더 참조
  • Git 연동: 커밋, PR 생성 자동화
  • Context 윈도우 관리 전략 학습

과제: Plan Mode로 기능 설계서 작성 → 구현 → Git 커밋까지 전 과정 수행

📅 Week 3 — Skills & 커스텀 명령어 난이도: 중급

목표: 반복 작업을 자동화하는 커스텀 Skills 만들기

  • .claude/skills/ 디렉토리 구성
  • 커스텀 슬래시 명령어 만들기 (/review, /test 등)
  • CLAUDE.md 고급 설정 (3000토큰 이내)
  • 프레임워크별 최적 설정 탐구
  • 팀 공통 Skills 세트 정의
  • Skills 공식 문서 정독

과제: 나만의 Skill 2개 이상 만들어서 공유

📅 Week 4 — Hooks & MCP 연동 난이도: 중급

목표: 이벤트 자동화 + 외부 도구 연결

  • Hooks 개념과 이벤트 유형 학습
  • PreToolUse/PostToolUse 훅 작성
  • 파일 저장 시 자동 포맷팅/린팅 훅
  • MCP 서버 개념 이해
  • Figma, Notion, DB 등 MCP 연동 실습
  • 커뮤니티 플러그인 탐색 & 설치

과제: MCP 서버 1개 연동 + 자동화 Hook 1개 작성

📅 Week 5 — 에이전트 & CI/CD 난이도: 고급

목표: 서브에이전트 구축 + GitHub Actions 연동

  • 서브에이전트 생성 (.claude/agents/)
  • 에이전트 팀 오케스트레이션
  • Headless 모드로 자동화 파이프라인
  • GitHub Actions와 Claude Code 통합
  • Code Review 자동화 설정
  • 비용 모니터링 & 최적화 전략

과제: GitHub Actions + Claude Code로 PR 자동 리뷰 파이프라인 구축

📅 Week 6 — 실전 프로젝트 & 정리 난이도: 종합

목표: 실전 프로젝트 완수 + 팀 워크플로우 확립

  • 실전 프로젝트 1: 웹 앱 / API 서버 구축
  • 실전 프로젝트 2: 데이터 파이프라인 / 분석 도구
  • 팀 CLAUDE.md 표준 템플릿 작성

  • 보안 & 비용 관리 가이드라인 수립
  • 학습 내용 발표 & 회고 세미나

⌨️ 명령어 치트시트

카테고리별로 정리된 핵심 명령어. 탭으로 분류를 전환하세요.

🚀 대화형 실행

claude대화형 세션 시작
claude "메시지"원라이너 실행
claude --resume마지막 세션 이어서
claude --add-dir 경로추가 디렉토리 참조
claude --model opus모델 지정 시작

🤖 Headless 모드

claude -p "프롬프트"비대화형 실행 (자동화용)
claude -p "..." --output-format jsonJSON 출력
cat file | claude -p "분석해줘"파이프 입력
claude -p "..." > result.txt결과를 파일로 저장
