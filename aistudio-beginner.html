1. 바이브코딩이란?

바이브코딩(Vibe Coding)은 코딩을 직접 하지 않고, AI에게 일상 언어로 원하는 것을 설명하면 AI가 코드를 작성해주는 방식입니다.

  • 프로그래밍 언어를 몰라도 됩니다
  • "이런 웹페이지를 만들어줘"라고 말하면 AI가 만들어줍니다
  • 마치 디자이너에게 의뢰하듯이 대화로 진행합니다

왜 주목받고 있을까?

기존 개발 방식바이브코딩
프로그래밍 언어 학습 필요자연어(한국어)로 지시
수백 줄의 코드를 직접 작성AI가 코드를 자동 생성
디버깅에 많은 시간 소요AI가 오류를 자동 수정
개발자만 가능누구나 시작 가능

2. AI Studio 빌드란?

AI Studio 빌드는 Google AI Studio(aistudio.google.com)의 Build 탭에서 제공하는 웹 기반 프로토타입 빌더입니다. Gemini 2.5 Pro AI 모델과 대화하며 웹 프로토타입을 만들 수 있습니다.

AI Studio 빌드의 특징

장점설명
설치 불필요웹 브라우저만 있으면 바로 시작 가능
완전 무료Google 계정만 있으면 무료로 사용 가능
대화형 제작채팅하듯이 원하는 것을 요청
실시간 프리뷰만든 결과물을 즉시 확인 가능
한국어 프롬프트한국어로 요청해도 잘 동작
AI 모델Gemini 2.5 Pro — Google 최신 AI 모델 사용

💡 팁: AI Studio는 "이런 것도 만들 수 있구나!" 하는 느낌을 빠르게 체험하는 데 최적화된 도구입니다. 완성도 높은 웹사이트를 만들고 싶다면 이후에 러버블로 넘어가세요.

주요 기능

참고: AntiGravity와의 관계

AI Studio의 빌드 기능은 내부적으로 AntiGravity Agent 기술을 활용합니다. 사용자 입장에서는 신경 쓸 필요 없이 웹 브라우저에서 대화만 하면 됩니다. AntiGravity는 데스크톱 앱으로 더 복잡한 프로젝트를 다루는 도구이고, AI Studio 빌드는 같은 기술을 웹에서 간편하게 쓸 수 있도록 한 것입니다.

3. 용어 해설

교안에서 자주 나오는 용어들을 미리 알아두세요:

용어쉬운 설명

Deploy (배포)만든 웹페이지를 인터넷에 공개하는 것
Fork (포크)다른 사람의 프로젝트를 복사하여 내 것으로 가져오는 것

4. 사전 준비

AI Studio 빌드를 시작하기 위해 필요한 것은 딱 하나, Google 계정입니다.

4-1. Google 계정 준비

이미 Gmail을 사용하고 있다면 별도의 가입이 필요 없습니다.

  • Gmail을 사용 중이라면 → 바로 시작 가능
  • Gmail이 없다면 → accounts.google.com에서 무료 가입

4-2. AI Studio 접속

  1. 웹 브라우저에서 aistudio.google.com 접속
  2. Google 계정으로 로그인
  3. AI Studio 메인 화면이 나타납니다

4-3. 빌드 모드 진입

  1. AI Studio 상단 또는 좌측 메뉴에서 Build 탭을 찾습니다
  2. Build 탭을 클릭하면 빌드 모드로 진입합니다
  3. 프롬프트 입력창이 나타나면 준비 완료!
💡 팁: AI Studio의 화면은 자주 업데이트됩니다. Build 탭의 위치가 바뀔 수 있으니, 찾기 어려우면 화면 상단의 메뉴를 살펴보세요.

✅ 사전 준비 체크리스트

  • Google 계정(Gmail) 준비 완료
  • AI Studio (aistudio.google.com) 접속 확인
  • Build 탭 위치 확인
  • 만들고 싶은 웹페이지 아이디어 1개 이상 구상

5. 아이디어 구상하기

AI Studio 빌드에 바로 뛰어들기 전에, 무엇을 만들 것인지 먼저 정리하는 것이 중요합니다.

AI Studio에 적합한 프로젝트

AI Studio 빌드는 가벼운 프로토타입에 적합합니다. 한 페이지로 완성되는 간단한 웹페이지가 좋습니다.

적합한 프로젝트: 자기소개 페이지, 간단한 계산기/타이머, 할 일 목록, 이벤트 안내 페이지, 퀴즈, 간단한 게임 등

다른 도구가 더 적합한 프로젝트: 회원가입/로그인 서비스(→ 러버블), 데이터베이스 게시판(→ 러버블 + Supabase), 복잡한 멀티페이지 앱(→ 커서/안티그래비티)

프롬프트 아이디어 예시

아직 무엇을 만들지 정하지 못했다면, 아래 예시를 참고하세요:

6. 프롬프트 작성하기

AI Studio 빌드에서 좋은 결과물을 얻으려면, 구체적인 프롬프트가 핵심입니다.

효과적인 프롬프트 패턴

좋은 프롬프트에는 3가지 요소가 포함됩니다:

  1. 무엇을 만들 것인지 (기능 설명)
  2. 어떤 스타일인지 (디자인, 색상)
  3. 어떤 동작을 하는지 (인터랙션, 기능)

좋은 프롬프트 vs 나쁜 프롬프트

나쁜 프롬프트좋은 프롬프트
"웹사이트 만들어줘""카페 소개 원페이지를 만들어줘. 상단에 카페 이름, 중간에 메뉴, 하단에 위치 표시"
"이쁘게 해줘""배경색을 연한 베이지로, 제목 폰트를 크고 굵게, 카드 형태로 구분해줘"
"수정해줘""제목 글자 크기를 32px로 키우고, 색상을 파란색(#2196F3)으로 변경해줘"

프롬프트 작성 팁

💡 팁: 처음부터 완벽하게 쓰려고 하지 마세요. 간단하게 요청하고, 결과를 보면서 수정해 나가면 됩니다.
  • 한 번에 하나씩: 여러 기능을 한꺼번에 요청하면 결과가 엉킬 수 있습니다
  • 구체적으로: "좀 더 이쁘게" 대신 "배경을 연한 노란색으로, 글자를 더 크게"
  • 예시를 포함: "카드 형태로", "네이버처럼" 같은 참고 표현이 도움됩니다

반복 수정 요청 방법

AI Studio 빌드에서는 대화로 계속 수정을 요청할 수 있습니다:

[첫 요청] 간단한 할 일 목록 웹페이지를 만들어줘.
[수정 1] 할 일을 추가하는 입력창을 상단에 배치해줘.
[수정 2] 완료된 항목에 취소선을 표시하고, 삭제 버튼도 추가해줘.
[수정 3] 전체 배경색을 연한 민트색으로 바꿔줘.
💡 팁: 마음에 드는 결과가 나왔다면 코드를 복사해두세요. 이후 수정에서 원치 않은 변경이 생길 수 있습니다.

7. AI Studio 실습: 나만의 프로토타입 만들기

이제 실제로 AI Studio 빌드를 사용하여 웹 프로토타입을 만들어봅시다!

Step 1: AI Studio 접속하기

  1. 웹 브라우저에서 aistudio.google.com 접속
  2. Google 계정으로 로그인
  3. 상단 또는 좌측 메뉴에서 Build 탭 클릭
  4. 빌드 모드 화면이 나타나면 준비 완료
💡 팁: Chrome 브라우저를 사용하는 것을 추천합니다. 다른 브라우저에서는 일부 기능이 다르게 동작할 수 있습니다.

Step 2: 프롬프트 입력하기

프롬프트 입력창에 만들고 싶은 것을 설명합니다. 첫 번째 실습으로 간단한 자기소개 페이지를 만들어봅시다:

자기소개 웹페이지를 만들어줘.

[구성]
- 상단: 이름 "김선생"과 직업 "교사"를 크게 표시
- 중간: 간단한 자기소개 문장 3줄
- 하단: 이메일 링크와 SNS 아이콘

[디자인]
- 깔끔한 카드 레이아웃
- 배경은 연한 회색, 카드는 흰색
- 둥근 모서리
- 모바일에서도 잘 보이게

입력 후 전송 버튼을 클릭하면 AI가 코드를 생성하기 시작합니다.

Step 3: 프리뷰 확인하기

AI가 코드를 생성하면 프리뷰 화면에서 결과를 바로 확인할 수 있습니다. 레이아웃, 텍스트, 색상, 모바일 화면까지 꼼꼼히 확인하세요.

💡 팁: 프리뷰에서 보이는 것이 실제 웹페이지와 동일합니다.

Step 4: 수정 반복하기

결과물이 완벽하지 않아도 괜찮습니다. 대화를 이어가며 수정을 요청하세요:

"이름 부분의 글자를 더 크게 해줘"
"배경색을 연한 파란색으로 바꿔줘"
"자기소개 문장 아래에 취미 목록을 추가해줘"

수정 요청 시 주의사항: 한 번에 하나씩, 구체적으로, 마음에 들면 코드를 백업해두세요.

💡 팁: AI와 대화를 너무 많이 이어가면 Context Window를 넘어서 이전 내용을 잊을 수 있습니다. 큰 변경이 필요하면 새 대화를 시작하세요.

8. 프로젝트 공개하기

완성된 프로토타입을 다른 사람에게 보여주는 방법입니다.

방법 1: 공유 링크 생성

AI Studio 빌드에서 완성된 프로젝트는 공유 링크를 통해 다른 사람에게 보여줄 수 있습니다.

  1. 빌드 화면에서 공유 또는 Deploy 버튼 찾기
  2. 링크를 생성하면 URL이 만들어집니다
  3. 이 URL을 공유하면 끝!

방법 2: 코드 다운로드 (HTML/CSS/JS)

AI가 생성한 코드를 직접 다운로드하여 활용할 수 있습니다.

  1. 생성된 코드를 전체 선택하여 복사
  2. 텍스트 편집기(메모장 등)에 붙여넣기
  3. index.html 파일명으로 저장
  4. 파일을 더블클릭하면 웹 브라우저에서 열립니다
💡 팁: 코드를 다운로드해두면 인터넷 없이도 결과물을 확인할 수 있고, 나중에 다른 도구에서 이어서 수정할 수도 있습니다.

방법 3: GitHub에 업로드하기 (선택)

코드를 체계적으로 관리하고 싶다면 GitHub에 업로드할 수 있습니다.

  1. github.com에서 New Repository 생성
  2. Upload files → index.html 파일 드래그 앤 드롭 → Commit changes
  3. GitHub Pages 활성화: Settings → Pages → Branch를 main 선택 → Save
  4. https://사용자이름.github.io/저장소이름 으로 접속 가능
이 방법은 GitHub에 익숙해진 후에 시도해도 됩니다. 처음에는 방법 1이나 2로 충분합니다.

10. 문제가 생겼을 때 (트러블슈팅)

"빌드 탭이 안 보여요"
  • AI Studio 화면이 업데이트되어 메뉴 위치가 바뀌었을 수 있습니다 — 상단/좌측 메뉴를 확인하세요
  • 브라우저를 새로고침(F5)하거나 다시 로그인해보세요
  • Chrome 브라우저 사용을 추천합니다
"프리뷰가 표시되지 않아요"
  • AI가 아직 코드를 생성 중일 수 있습니다 — 잠시 기다려보세요
  • 프리뷰 패널이 접혀있거나, 브라우저 팝업 차단이 원인일 수 있습니다
  • 코드를 복사하여 로컬 HTML 파일로 저장한 후 열어보는 것도 방법입니다
"응답이 너무 느려요"
  • 사용량이 많은 시간대이거나, 프롬프트가 너무 복잡할 수 있습니다
  • 요청을 단순화하고, 인터넷 연결을 확인한 후 다시 시도하세요
"생성된 코드에 오류가 있어요"
  • AI에게 구체적으로 오류를 알려주세요: "하단의 버튼이 작동하지 않아"
  • "코드를 검증하고 오류가 있으면 수정해줘"라고 요청해보세요
  • 문제가 반복되면 새 대화를 시작하여 처음부터 다시 만드는 것이 빠를 수 있습니다
"프로젝트를 저장하고 싶어요"
  • 확실한 보관을 위해 코드를 직접 복사하여 .html 파일로 저장하세요
  • GitHub에 업로드하면 버전 관리까지 가능합니다

11. 자주 묻는 질문 (FAQ)

Q1. AI Studio 빌드는 완전 무료인가요?

네, Google 계정만 있으면 무료입니다. 다만 일일 API 호출 한도가 있어서 하루 사용 횟수에는 제한이 있습니다. 일반적인 학습과 프로토타이핑에는 충분한 수준입니다.

Q2. 러버블(Lovable)과 뭐가 다른가요?

AI Studio는 가벼운 프로토타입 빌더, 러버블은 풀스택 웹앱 빌더입니다. AI Studio에서 아이디어를 빠르게 테스트하고, 본격적인 웹사이트는 러버블로 만드는 것을 추천합니다.

Q3. 한국어로 프롬프트 써도 되나요?

네! Gemini 2.5 Pro는 한국어를 잘 이해합니다. 한국어 프롬프트로 한국어 웹페이지를 만들 수 있습니다.

Q4. 코딩을 전혀 몰라도 사용할 수 있나요?

네, 자연어로 설명하면 AI가 코드를 자동 생성합니다. 바이브코딩 도구 중 가장 진입 장벽이 낮은 도구입니다.

Q5. 만든 프로젝트를 저장할 수 있나요?

생성된 코드를 복사하여 .html 파일로 저장할 수 있습니다. 대화 히스토리도 유지되므로 이전 작업을 이어갈 수 있습니다. 중요한 결과물은 반드시 로컬 파일로 백업하세요.

Q6. 코드를 내보낼 수 있나요?

네, HTML/CSS/JavaScript 코드를 복사하여 어디서든 사용할 수 있습니다. 다른 도구(러버블, 커서, VS Code 등)에서 이어서 수정도 가능합니다.

Q7. 모바일에서도 사용할 수 있나요?

접속은 가능하지만, PC 환경이 훨씬 편리합니다. 모바일에서는 화면이 작아 작업 효율이 떨어질 수 있습니다.

Q8. 여러 프로젝트를 만들 수 있나요?

네, 새 대화를 시작하면 새 프로젝트를 만들 수 있습니다. 대화 단위로 프로젝트가 관리됩니다.
