## 1. 구글 생태계란?

### 학습 목표

- 구글이 제공하는 주요 서비스와 각각의 용도를 파악한다
- 구글 생태계의 핵심 강점을 이해한다
- 무료와 유료(Google Workspace) 플랜의 차이를 안다

### 1-1. 주요 서비스 한눈에 보기

구글은 하나의 계정으로 수십 가지 서비스를 무료로 이용할 수 있는 거대한 디지털 생태계입니다. 아래는 가장 많이 쓰이는 핵심 서비스입니다.

| 서비스 | 아이콘 | 설명 | 주요 용도 |
|--------|--------|------|----------|
| **Gmail** | ✉️ | 이메일 서비스 | 메일 송수신, 일정 연동 |
| **Google Drive** | 📁 | 클라우드 저장소 | 파일 저장, 공유, 협업 |
| **Google Docs** | 📝 | 문서 작성 도구 | 보고서, 기안문, 회의록 |
| **Google Sheets** | 📊 | 스프레드시트 | 데이터 관리, 계산, 차트 |
| **Google Slides** | 📽️ | 프레젠테이션 | 발표 자료 제작 |
| **Google Forms** | 📋 | 설문조사 도구 | 설문, 퀴즈, 신청서 |
| **Google Calendar** | 📅 | 일정 관리 | 스케줄, 미팅, 알림 |
| **Google Meet** | 🎥 | 화상 회의 | 온라인 미팅, 수업 |

### 1-2. 구글 생태계의 핵심 강점

구글 생태계가 전 세계적으로 사랑받는 이유는 다음 네 가지 강점 때문입니다.

**1. 무료** — 대부분의 핵심 서비스를 무료로 이용할 수 있습니다.
- Gmail 15GB, Drive 15GB, Docs/Sheets/Slides 무제한 생성
- 추가 비용 없이 팀 협업 기능까지 제공

**2. 통합** — 하나의 계정으로 모든 서비스가 연결됩니다.
- Gmail에서 받은 첨부파일이 자동으로 Drive에 저장
- Docs에서 작성한 문서를 Slides에서 바로 참조
- Calendar 일정에 Meet 링크 자동 생성

**3. 클라우드** — 인터넷만 있으면 어디서든 접근할 수 있습니다.
- 설치할 프로그램이 없습니다 (브라우저에서 모두 실행)
- 자동 저장 — 별도로 저장 버튼을 누를 필요가 없습니다
- PC, 스마트폰, 태블릿 어디서든 동일한 파일에 접근

**4. 협업** — 여러 사람이 동시에 같은 문서를 편집할 수 있습니다.
- 실시간 공동 편집 (다른 사람의 커서가 보임)
- 댓글, 제안 모드로 피드백 교환
- 공유 링크 하나로 누구에게나 전달 가능

### 1-3. 무료 vs 유료(Google Workspace) 비교

| 항목 | 무료 (개인 계정) | Google Workspace (유료) |
|------|-----------------|----------------------|
| 이메일 | Gmail (@gmail.com) | 맞춤 도메인 (@회사명.com) |
| 저장 용량 | 15GB (Gmail+Drive+Photos 합산) | 30GB ~ 무제한 (플랜별 상이) |
| 관리 콘솔 | 없음 | 조직 관리, 사용자 관리 가능 |
| 보안 | 기본 | 고급 보안, DLP, 감사 로그 |
| 지원 | 커뮤니티 포럼 | 24/7 기술 지원 |
| Gemini | 기본 (Flash 모델) | Advanced (Ultra 모델) |
| 가격 | 무료 | 월 $7.20~ (플랜별 상이) |

> **팁**: 개인 사용자라면 무료 계정으로도 충분합니다. 학교·기업 등 조직 단위로 사용할 때 Google Workspace를 고려하세요.

### 1-4. 구글 앱 생태계 전체 조감도

구글 서비스들은 서로 긴밀하게 연결되어 있습니다. 아래는 서비스 간 연동 흐름입니다.

```
                    ┌─────────────┐
                    │   Gmail     │
                    │  (이메일)    │
                    └──────┬──────┘
                           │ 첨부파일 자동 저장
                    ┌──────▼──────┐
        ┌──────────►│ Google Drive │◄──────────┐
        │           │ (클라우드)    │           │
        │           └──────┬──────┘           │
        │                  │                   │
   ┌────┴────┐    ┌───────┼───────┐    ┌─────┴─────┐
   │  Docs   │    │  Sheets      │    │  Slides   │
   │ (문서)   │    │ (스프레드시트) │    │ (발표)    │
   └────┬────┘    └───────┬───────┘    └─────┬─────┘
        │                 │                   │
        │          ┌──────▼──────┐            │
        └─────────►│   Forms     │◄───────────┘
                   │  (설문조사)   │
                   └──────┬──────┘
                          │ 응답 데이터
                   ┌──────▼──────┐
                   │ NotebookLM  │◄──── Gemini (AI 지원)
                   │ (AI 분석)    │
                   └─────────────┘
```

> **팁**: 구글 서비스의 가장 큰 매력은 이 연동입니다. Forms의 응답이 Sheets로 자동 저장되고, Docs의 내용을 NotebookLM에서 분석할 수 있습니다.

## 2. Google 계정과 Workspace

### 학습 목표

- 구글 계정을 만들고 기본 설정을 관리할 수 있다
- 2단계 인증을 설정하여 계정 보안을 강화할 수 있다
- 계정 복구 방법을 알고 대비할 수 있다

### 2-1. 구글 계정 만들기

구글 계정은 모든 구글 서비스의 출발점입니다. 이미 Gmail 주소가 있다면 별도 가입 없이 바로 사용할 수 있습니다.

**새 계정 만들기**:

```
1. accounts.google.com 접속
2. "계정 만들기" 클릭
3. 이름 입력 (성, 이름)
4. 원하는 Gmail 주소 선택 (예: myname@gmail.com)
5. 비밀번호 설정 (8자 이상, 영문+숫자+특수문자 조합 권장)
6. 전화번호 인증 (선택이지만 보안을 위해 권장)
7. 복구 이메일 설정 (비밀번호 분실 시 필요)
8. 생년월일, 성별 입력
9. 약관 동의 후 완료
```

> **주의**: 이미 구글 계정이 있다면 새로 만들 필요 없습니다. 하나의 계정으로 모든 서비스를 이용할 수 있습니다.

### 2-2. 계정 관리하기

계정 설정은 [myaccount.google.com](https://myaccount.google.com)에서 관리합니다.

| 메뉴 | 설명 | 확인 사항 |
|------|------|----------|
| **개인정보** | 이름, 생일, 프로필 사진 | 프로필 사진 설정 |
| **데이터 및 개인 정보 보호** | 활동 기록, 광고 설정 | 웹 활동 기록 관리 |
| **보안** | 비밀번호, 2단계 인증 | 보안 점검 실행 |
| **사람 및 공유** | 연락처, 공유 설정 | 공유 기본값 확인 |
| **결제 및 구독** | Google One, 결제 수단 | 구독 상태 확인 |

### 2-3. 2단계 인증 설정 (필수!)

2단계 인증은 비밀번호가 유출되더라도 계정을 보호하는 가장 중요한 보안 설정입니다.

**설정 방법**:

```
1. myaccount.google.com → 보안 탭
2. "Google에 로그인하는 방법" 섹션
3. "2단계 인증" 클릭
4. "시작하기" 클릭
5. 비밀번호 재입력
6. 인증 방법 선택:
   - Google 메시지 (추천): 스마트폰에 알림이 옴
   - 인증 앱: Google Authenticator 등 사용
   - 문자 메시지: SMS로 인증 코드 수신
   - 보안 키: 물리적 USB 키 사용
7. 백업 코드 저장 (반드시 안전한 곳에 보관!)
```

> **⚠️ 중요**: 백업 코드는 스마트폰을 분실했을 때 계정에 접근하는 유일한 방법입니다. 반드시 인쇄하거나 안전한 곳에 저장하세요.

### 2-4. Google Workspace 개요

| 구분 | 개인 계정 | Google Workspace |
|------|----------|-----------------|
| 대상 | 개인 사용자 | 기업, 학교, 단체 |
| 이메일 | @gmail.com | @회사명.com (맞춤 도메인) |
| 관리 | 본인만 관리 | IT 관리자가 조직 전체 관리 |
| 비용 | 무료 | 유료 (교육기관은 무료 가능) |
| 특징 | 개인 파일 관리 | 공유 드라이브, 관리 콘솔, 감사 로그 |

> **팁**: 학교에서 Google Workspace for Education을 사용한다면 `@학교도메인.kr` 형태의 계정이 제공됩니다. 이 계정은 무제한 저장 용량 등 추가 혜택이 있을 수 있습니다.

### 2-5. 보안 설정 필수 항목

계정을 만든 후 반드시 확인해야 할 보안 설정입니다.

```
✅ 체크리스트:
□ 2단계 인증 활성화
□ 복구 전화번호 등록
□ 복구 이메일 등록
□ 보안 점검(Security Checkup) 실행
□ 연결된 타사 앱 검토
□ 최근 보안 활동 확인
□ 백업 코드 저장
```

**보안 점검 실행 방법**:

```
1. myaccount.google.com → 보안 탭
2. "보안 점검" 또는 "계정을 안전하게 보호하세요" 클릭
3. 안내에 따라 항목별로 점검
4. 노란색/빨간색 경고 항목은 즉시 조치
```

### 2-6. 계정 복구 방법

비밀번호를 잊어버렸거나 계정에 접근할 수 없을 때의 복구 방법입니다.

```
1. accounts.google.com/signin/recovery 접속
2. 이메일 주소 입력
3. 복구 옵션 중 하나 선택:
   - 등록된 전화번호로 인증 코드 수신
   - 복구 이메일로 인증 코드 수신
   - 백업 코드 입력
   - 최근 사용한 비밀번호 입력
4. 본인 확인 후 새 비밀번호 설정
```

> **팁**: 복구 전화번호와 복구 이메일을 미리 등록해두는 것이 가장 중요합니다. 등록하지 않으면 계정 복구가 매우 어려워집니다.

### 실습 과제: 계정 보안 점검

```
[실습] 내 구글 계정 보안 강화하기

1. myaccount.google.com 접속
2. 보안 탭으로 이동
3. 2단계 인증이 활성화되어 있는지 확인
   - 비활성화 상태라면 지금 바로 설정
4. 복구 전화번호와 복구 이메일 등록 확인
5. "보안 점검" 실행하여 모든 항목 초록색으로 만들기
6. 연결된 타사 앱 목록 검토 → 사용하지 않는 앱 삭제

⏱️ 예상 소요: 10~15분
```

## 3. Google Docs

### 학습 목표

- Google Docs에서 새 문서를 만들고 기본 서식을 적용할 수 있다
- 실시간 협업 기능(공유, 댓글, 제안 모드)을 사용할 수 있다
- 문서를 다양한 형식(PDF, Word)으로 내보낼 수 있다

### 3-1. 문서 작성 기초

Google Docs는 클라우드 기반 문서 작성 도구입니다. Microsoft Word와 비슷하지만, 설치 없이 브라우저에서 바로 사용할 수 있고, 자동 저장됩니다.

**새 문서 만들기**:

```
방법 1: 직접 접속
  docs.google.com → "빈 문서" 클릭

방법 2: Google Drive에서
  drive.google.com → "새로 만들기" → "Google 문서"

방법 3: 주소창에 입력
  브라우저 주소창에 docs.new 입력 → 바로 새 문서 생성
```

> **팁**: `docs.new`를 브라우저 주소창에 입력하면 즉시 새 문서가 열립니다. `sheets.new`, `slides.new`, `forms.new`도 마찬가지입니다!

### 3-2. 기본 서식 적용

| 서식 | 방법 | 단축키 |
|------|------|--------|
| **굵게** | 도구 모음의 B 클릭 | `Ctrl + B` |

## 4. Google Sheets

### 학습 목표

- Google Sheets의 기본 인터페이스와 셀/행/열 개념을 이해한다
- 핵심 함수 10가지를 사용하여 데이터를 계산할 수 있다
- 차트를 만들고, 필터와 정렬로 데이터를 정리할 수 있다

### 4-1. 스프레드시트 기초

Google Sheets는 데이터를 표 형태로 관리하고 계산하는 도구입니다. Microsoft Excel과 유사합니다.

**기본 용어**:

| 용어 | 설명 | 예시 |
|------|------|------|
| **셀(Cell)** | 데이터를 입력하는 가장 작은 단위 (칸) | A1, B3, C10 |
| **행(Row)** | 가로줄 (숫자로 표시) | 1행, 2행, 3행... |
| **열(Column)** | 세로줄 (알파벳으로 표시) | A열, B열, C열... |
| **시트(Sheet)** | 하나의 탭 (하단에 탭으로 표시) | Sheet1, Sheet2 |
| **범위(Range)** | 여러 셀의 묶음 | A1:C10 (A1부터 C10까지) |
| **수식(Formula)** | 계산을 위한 식 (= 으로 시작) | =A1+B1 |

**새 스프레드시트 만들기**:

```
방법 1: sheets.google.com → "빈 스프레드시트" 클릭
방법 2: 브라우저 주소창에 sheets.new 입력
방법 3: Drive → 새로 만들기 → Google 스프레드시트
```

### 4-2. 핵심 함수 10선

함수는 `=함수이름(인수)` 형태로 셀에 입력합니다.

**1. SUM — 합계**:

```
=SUM(A1:A10)          → A1부터 A10까지의 합계
=SUM(A1, B1, C1)      → A1, B1, C1 세 셀의 합계
=SUM(A1:A10, C1:C10)  → 두 범위의 합계
```

**2. AVERAGE — 평균**:

```
=AVERAGE(A1:A10)      → A1~A10의 평균값
```

**3. COUNT / COUNTA — 개수 세기**:

```
=COUNT(A1:A10)        → 숫자가 있는 셀의 개수
=COUNTA(A1:A10)       → 비어있지 않은 셀의 개수 (텍스트 포함)
```

**4. MAX / MIN — 최대값 / 최소값**:

```
=MAX(A1:A10)          → 최대값
=MIN(A1:A10)          → 최소값
```

**5. IF — 조건 판단**:

```
=IF(조건, 참일 때 값, 거짓일 때 값)

=IF(A1>=60, "합격", "불합격")
=IF(B1>100, B1*0.9, B1)  → 100 초과면 10% 할인
```

**6. VLOOKUP — 세로 검색**:

```
=VLOOKUP(검색값, 범위, 열번호, FALSE)

=VLOOKUP("사과", A1:C10, 3, FALSE)
→ A열에서 "사과"를 찾아 같은 행의 C열(3번째 열) 값 반환
```

**7. CONCATENATE — 텍스트 합치기**:

```
=CONCATENATE(A1, " ", B1)    → A1과 B1 사이에 공백 넣어 합치기
=A1 & " " & B1               → 같은 결과 (& 연산자 사용)
```

**8. TODAY / NOW — 오늘 날짜 / 현재 시간**:

```
=TODAY()              → 오늘 날짜 (예: 2026-03-09)
=NOW()                → 현재 날짜와 시간
=DATEDIF(A1, TODAY(), "D")  → A1 날짜로부터 오늘까지 일수
```

**9. COUNTIF — 조건부 개수 세기**:

```
=COUNTIF(A1:A10, "합격")          → "합격"인 셀 개수
=COUNTIF(B1:B10, ">80")          → 80 초과인 셀 개수
=COUNTIF(C1:C10, "서울")          → "서울"인 셀 개수
```

**10. SUMIF — 조건부 합계**:

```
=SUMIF(조건범위, 조건, 합계범위)

=SUMIF(A1:A10, "서울", C1:C10)
→ A열이 "서울"인 행의 C열 값만 합산
```

> **팁**: 함수를 외울 필요 없습니다! 셀에 `=`을 입력하고 함수 이름의 처음 몇 글자만 치면 자동완성 목록이 나타납니다.

### 4-3. 서식 지정

**조건부 서식 기초**:

특정 조건에 따라 셀의 색상을 자동으로 변경할 수 있습니다.

```
1. 범위 선택 (예: B2:B20)
2. 메뉴 → 서식 → 조건부 서식
3. 조건 설정:
   - "보다 크거나 같음" → 90 → 초록 배경
   - "보다 작음" → 60 → 빨강 배경
4. "완료" 클릭
```

**셀 병합**:

```
1. 병합할 셀 범위 선택 (예: A1:C1)
2. 도구 모음의 "셀 병합" 아이콘 클릭
3. 병합 유형 선택:
   - 모두 병합: 선택한 모든 셀을 하나로
   - 가로로 병합: 각 행 내에서만 병합
   - 세로로 병합: 각 열 내에서만 병합
```

**테두리**:

```
1. 범위 선택
2. 도구 모음의 "테두리" 아이콘 (격자 모양) 클릭
3. 원하는 테두리 스타일 선택 (전체, 외곽만, 안쪽만 등)
4. 테두리 색상과 굵기도 변경 가능
```

### 4-4. 차트 만들기

데이터를 시각적으로 표현하면 이해하기 쉽습니다.

```
1. 차트에 넣을 데이터 범위 선택 (헤더 행 포함)
2. 메뉴 → 삽입 → 차트
3. 차트 편집기에서 차트 유형 선택:
   - 막대형 차트: 항목 간 비교에 적합
   - 원형 차트: 비율/구성 표현에 적합
   - 꺾은선형 차트: 시간에 따른 변화 표현에 적합
   - 분산형 차트: 두 변수 간 관계 표현에 적합
4. "맞춤설정" 탭에서 제목, 색상, 라벨 등 수정
5. 차트를 드래그하여 원하는 위치로 이동
```

| 차트 유형 | 적합한 상황 | 예시 |
|----------|------------|------|
| 막대형 | 항목별 비교 | 과목별 성적, 월별 매출 |
| 원형 | 비율/구성 | 지출 비율, 시장 점유율 |
| 꺾은선형 | 시간 변화 추이 | 기온 변화, 주가 추이 |
| 분산형 | 상관관계 | 키와 몸무게, 공부시간과 성적 |

### 4-5. 필터와 정렬

**정렬**:

```
1. 정렬 기준이 될 열의 아무 셀 클릭
2. 메뉴 → 데이터 → 시트 정렬 기준
3. 선택:
   - A → Z 정렬 (오름차순): 작은 수부터, 가나다 순
   - Z → A 정렬 (내림차순): 큰 수부터, 역순
```

**필터**:

```
1. 데이터 범위의 아무 셀 클릭
2. 메뉴 → 데이터 → 필터 만들기
3. 각 열 헤더에 드롭다운 화살표(▼)가 나타남
4. 화살표 클릭 → 보고 싶은 값만 체크
5. 필터 해제: 메뉴 → 데이터 → 필터 삭제
```

> **팁**: "필터 보기"를 사용하면 다른 사용자에게 영향 주지 않고 본인만의 필터를 적용할 수 있습니다. (데이터 → 필터 보기 → 새 필터 보기 만들기)

### 실습 과제: 가계부 만들기

```
[실습] 1주일 가계부 만들기

1. sheets.new로 새 스프레드시트 생성
2. 제목: "3월 첫째 주 가계부"

3. 헤더 행 만들기 (1행):
   A1: 날짜 | B1: 분류 | C1: 내용 | D1: 수입 | E1: 지출

4. 데이터 입력 (2행~):
   예시 데이터 7~10건 입력
   - 분류: 식비, 교통비, 문화, 급여, 기타 등

5. 함수 활용 (아래 행에):
   - =SUM(D2:D20)         → 총 수입
   - =SUM(E2:E20)         → 총 지출
   - =SUM(D2:D20)-SUM(E2:E20)  → 잔액
   - =AVERAGE(E2:E20)     → 평균 지출
   - =MAX(E2:E20)         → 최대 지출
   - =COUNTIF(B2:B20, "식비")   → 식비 건수
   - =SUMIF(B2:B20, "식비", E2:E20) → 식비 합계

6. 조건부 서식:
   - 지출이 50,000원 이상인 셀을 빨간 배경으로

7. 차트 만들기:
   - 분류별 지출 합계 원형 차트

⏱️ 예상 소요: 25~30분
```

1. 메뉴 → 슬라이드 → 테마 변경
2. 우측에 테마 목록이 나타남
3. 원하는 테마 클릭 → 모든 슬라이드에 자동 적용
4. "테마 가져오기"로 외부 테마도 사용 가능
```

**마스터 슬라이드(레이아웃)**:

슬라이드를 추가할 때 용도에 맞는 레이아웃을 선택할 수 있습니다.

| 레이아웃 | 용도 |
|----------|------|
| 제목 슬라이드 | 첫 페이지, 큰 제목과 부제 |
| 섹션 헤더 | 새로운 장(섹션)의 시작 |
| 제목과 본문 | 가장 일반적, 제목 + 내용 |
| 제목만 | 제목 아래 자유 배치 |
| 빈 슬라이드 | 완전 자유 레이아웃 |
| 두 열 | 좌우 비교, 대조 |

### 5-3. 콘텐츠 삽입

**텍스트 삽입**:

```
1. 슬라이드 빈 공간 클릭
2. 텍스트 상자에 내용 입력
3. 새 텍스트 상자 추가: 삽입 → 텍스트 상자 → 드래그
```

**이미지 삽입**:

```
메뉴 → 삽입 → 이미지

옵션:
  - 컴퓨터에서 업로드
  - 웹 검색 (Google 이미지 검색)
  - Drive에서 선택
  - 사진 (Google Photos)
  - URL로 삽입
  - 카메라로 촬영
```

**도형 삽입**:

```
1. 메뉴 → 삽입 → 도형
2. 카테고리 선택: 도형, 화살표, 설명선, 수학 기호
3. 원하는 도형 클릭 후 슬라이드에서 드래그
4. 도형 안에 텍스트 입력 가능 (더블 클릭)
5. 색상, 테두리, 그림자 등 서식 변경 가능
```

### 5-4. 전환 효과와 애니메이션

**슬라이드 전환**:

```
1. 슬라이드 선택
2. 메뉴 → 슬라이드 → 전환
3. 우측 패널에서 전환 효과 선택:
   - 없음, 디졸브, 페이드, 슬라이드(방향별), 플립, 큐브 등
4. 속도 조절 (느리게/보통/빠르게)
5. "모든 슬라이드에 적용" 클릭 → 전체 적용
```

**개체 애니메이션**:

```
1. 애니메이션을 적용할 개체(텍스트, 이미지 등) 클릭
2. 메뉴 → 삽입 → 애니메이션 (또는 우측 패널)
3. 유형 선택:
   - 나타내기: 페이드 인, 날아오기, 확대 등
   - 사라지기: 페이드 아웃, 날아가기, 축소 등
   - 강조: 회전, 깜빡임 등
4. 시작 조건: 클릭 시, 이전 효과와 함께, 이전 효과 후
5. 속도 조절
```

> **팁**: 전환 효과와 애니메이션은 적당히 사용하는 것이 좋습니다. 너무 많으면 발표 내용에 집중하기 어려워집니다. 프로페셔널한 발표에서는 "페이드" 정도만 사용하는 경우가 많습니다.

### 5-5. 발표자 모드 활용

발표 시 청중에게는 슬라이드만 보이고, 발표자에게는 메모와 다음 슬라이드가 보이는 기능입니다.

```
발표자 모드 시작:
1. 메뉴 → 슬라이드쇼 → 발표자 보기 (Ctrl + Shift + Enter가 아닌 Ctrl + F5)
   또는 우측 상단 "슬라이드쇼" 버튼 옆 ▼ → "발표자 보기"

발표자 화면에서 보이는 것:
  - 현재 슬라이드
  - 다음 슬라이드 미리보기
  - 발표자 노트
  - 경과 시간 / 현재 시간
  - 레이저 포인터 도구
  - 청중 질문 도구 (Q&A)
```

**Q&A(질문) 기능**:

```
1. 발표자 보기에서 "Q&A 시작" 클릭
2. 청중에게 URL 공유 (또는 슬라이드 상단에 표시)
3. 청중이 웹에서 질문 입력
4. 발표자 화면에 실시간으로 질문이 표시
5. 인기 질문은 투표로 상위 노출
```

### 실습 과제: 자기소개 프레젠테이션

```
[실습] "나를 소개합니다" 프레젠테이션 만들기

1. slides.new로 새 프레젠테이션 생성
2. 테마 하나 선택

3. 슬라이드 구성 (5~7장):
   - 1장: 제목 슬라이드 (이름, 한 줄 소개)
   - 2장: 기본 정보 (직업/학교, 거주지 등)
   - 3장: 관심사/취미 (이미지 포함)
   - 4장: 좋아하는 것 (도형과 아이콘 활용)
   - 5장: 올해의 목표 (목록 형태)
   - 6장: Q&A / 감사 인사

4. 각 슬라이드에 적용:
   - 이미지 최소 3장 삽입
   - 도형 또는 아이콘 활용
   - 발표자 노트에 대본 작성

5. 전환 효과 적용 (페이드 또는 디졸브 권장)
6. 발표자 모드로 리허설 진행

⏱️ 예상 소요: 25~30분
```

## 6. Google Forms

### 학습 목표

- Google Forms로 다양한 유형의 설문조사를 만들 수 있다
- 섹션과 논리 분기를 활용하여 맞춤형 설문을 구성할 수 있다
- 응답 결과를 확인하고 스프레드시트로 내보낼 수 있다

### 6-1. 설문조사 만들기

Google Forms는 설문조사, 퀴즈, 신청서 등을 간편하게 만들 수 있는 도구입니다.

```
새 설문 만들기:
1. forms.google.com → "빈 양식" 클릭
   또는 브라우저 주소창에 forms.new 입력
2. 제목 입력 (예: "워크숍 만족도 설문")
3. 설명 입력 (예: "본 설문은 약 3분 소요됩니다")
```

### 6-2. 문항 유형

Google Forms에서 사용할 수 있는 문항 유형입니다.

| 유형 | 아이콘 | 설명 | 적합한 상황 |
|------|--------|------|------------|
| **단답형** | 짧은 줄 | 한 줄 텍스트 입력 | 이름, 이메일 |
| **장문형** | 긴 줄 | 여러 줄 텍스트 입력 | 의견, 소감 |
| **객관식** | ○ | 하나만 선택 | 성별, 만족도 |
| **체크박스** | ☑ | 여러 개 선택 가능 | 관심 분야, 참석 요일 |
| **드롭다운** | ▼ | 목록에서 하나 선택 | 학년, 지역 |
| **파일 업로드** | 📎 | 파일 첨부 | 과제 제출, 사진 첨부 |
| **선형 배율** | 1──5 | 숫자 척도 (1~10) | 만족도, 동의 정도 |
| **객관식 그리드** | 표 ○ | 행렬 형태 객관식 | 여러 항목 평가 |
| **체크박스 그리드** | 표 ☑ | 행렬 형태 체크박스 | 시간표, 일정 선택 |
| **날짜** | 📅 | 날짜 선택 | 생년월일, 희망 날짜 |
| **시간** | 🕐 | 시간 선택 | 희망 시간 |

**문항 추가 방법**:

```
1. 우측의 + (질문 추가) 아이콘 클릭
2. 문항 유형 선택
3. 질문 텍스트 입력
4. 옵션/답변 보기 추가
5. "필수" 토글을 켜면 반드시 응답해야 하는 문항이 됨
```

### 6-3. 섹션 나누기와 논리 분기

**섹션 나누기**:

긴 설문을 여러 페이지로 나눌 수 있습니다.

```
1. 우측 도구 모음에서 "섹션 추가" (═ 아이콘) 클릭
2. 섹션 제목과 설명 입력
3. 각 섹션이 별도 페이지로 표시됨
```

**논리 분기 (조건부 이동)**:

응답에 따라 다른 섹션으로 이동하게 만들 수 있습니다.

```
예시: "프로그래밍 경험이 있나요?"
  - "예" → 섹션 3 (경험자 질문)으로 이동
  - "아니오" → 섹션 4 (초보자 질문)으로 이동

설정 방법:
1. 객관식 문항 생성
2. 문항 우측 하단 ⋮ → "응답을 기반으로 섹션 이동" 선택
3. 각 옵션 옆에 이동할 섹션 선택
```

> **팁**: 논리 분기를 사용하면 응답자에게 꼭 필요한 질문만 보여줄 수 있어 설문 완료율이 높아집니다.

### 6-4. 응답 수집과 스프레드시트 연동

**응답 확인**:

```
1. Forms 상단의 "응답" 탭 클릭
2. 세 가지 보기 방식:
   - 요약: 전체 응답의 차트 요약
   - 질문별: 각 질문의 응답 분포
   - 개별: 응답자별 전체 답변
```

**스프레드시트 연동**:

```
1. "응답" 탭에서 스프레드시트 아이콘 (📊) 클릭
2. "새 스프레드시트 만들기" 또는 "기존 스프레드시트 선택"
3. 자동으로 Google Sheets에 응답 데이터가 기록됨
4. 새 응답이 들어올 때마다 자동으로 행이 추가됨
```

> **팁**: 스프레드시트에 연결하면 Sheets의 함수와 차트 기능을 활용하여 더 상세한 분석이 가능합니다.

### 6-5. 퀴즈 모드 활용

Google Forms를 시험이나 퀴즈로 활용할 수 있습니다.

```
퀴즈 모드 설정:
1. 설정(톱니바퀴) → "퀴즈로 설정" 활성화
2. 옵션 설정:
   - 점수 공개: 즉시 / 수동 검토 후
   - 응답자가 볼 수 있는 항목: 맞힌 문제, 정답, 점수
3. 각 문항에서 "답안" 클릭하여 정답과 배점 설정
4. 오답에 대한 피드백 메시지 입력 가능
```

### 실습 과제: 만족도 설문 만들기

```
[실습] "워크숍 만족도 설문" 만들기

1. forms.new로 새 양식 생성
2. 제목: "○○ 워크숍 만족도 설문"
3. 설명: "본 설문은 약 3분 소요됩니다. 솔직한 응답 부탁드립니다."

4. 섹션 1 — 기본 정보:
   - 이름 (단답형, 필수)
   - 소속 (드롭다운: 초등, 중등, 고등, 기타)
   - 교직 경력 (객관식: 5년 미만, 5~10년, 10~20년, 20년 이상)

5. 섹션 2 — 만족도 평가:
   - 전체 만족도 (선형 배율 1~5)
   - 항목별 평가 (객관식 그리드):
     행: 강사, 내용, 자료, 시설
     열: 매우 불만족, 불만족, 보통, 만족, 매우 만족
   - 가장 유익했던 내용 (체크박스, 복수 선택)

6. 섹션 3 — 서술형:
   - 좋았던 점 (장문형)
   - 개선 희망 사항 (장문형)
   - 다음 워크숍 희망 주제 (단답형)

7. 스프레드시트 연동 설정
8. 미리보기로 직접 응답 테스트

⏱️ 예상 소요: 20~25분
```

## 7. Google Drive

### 학습 목표

- Google Drive에 파일을 업로드하고 체계적인 폴더 구조를 만들 수 있다
- 공유 권한(뷰어, 댓글 작성자, 편집자)을 이해하고 설정할 수 있다
- 검색 기능과 오프라인 설정을 활용할 수 있다

### 7-1. 파일 업로드와 폴더 구조

Google Drive는 구글의 클라우드 저장소입니다. 모든 구글 문서가 자동으로 저장되며, 다른 파일도 업로드할 수 있습니다.

**파일 업로드**:

```
방법 1: 드래그 앤 드롭
  - 컴퓨터의 파일을 Drive 웹 화면으로 직접 끌어다 놓기

방법 2: 업로드 버튼
  - "새로 만들기" → "파일 업로드" 또는 "폴더 업로드"

방법 3: 데스크톱 앱
  - Google Drive for Desktop 설치 → 폴더 자동 동기화
```

**폴더 구조 만들기**:

```
효율적인 폴더 구조 예시:

📁 내 드라이브
├── 📁 업무
│   ├── 📁 2026년 1학기
│   │   ├── 📁 수업 자료
│   │   ├── 📁 평가 자료
│   │   └── 📁 회의록
│   └── 📁 2026년 2학기
├── 📁 개인
│   ├── 📁 가계부
│   ├── 📁 여행
│   └── 📁 자기개발
├── 📁 공유 프로젝트
│   ├── 📁 AI 연수
│   └── 📁 동아리
└── 📁 임시 (정리 필요)
```

> **팁**: 폴더 이름 앞에 번호를 붙이면 원하는 순서로 정렬할 수 있습니다. 예: `01_수업자료`, `02_평가자료`, `03_행정`.

### 7-2. 공유 권한 설정

| 권한 | 설명 | 사용 예시 |
|------|------|----------|
| **뷰어** | 보기만 가능 | 공지사항, 완성된 보고서 |
| **댓글 작성자** | 보기 + 댓글 가능 | 초안 피드백 요청 |
| **편집자** | 모든 편집 가능 | 공동 작업 문서 |

**공유 방법**:

```
방법 1: 특정 사용자에게 공유
  1. 파일/폴더 우클릭 → "공유"
  2. 이메일 주소 입력
  3. 권한 선택 (뷰어/댓글 작성자/편집자)
  4. 알림 메일 전송 여부 선택
  5. "보내기" 클릭

방법 2: 링크로 공유
  1. 파일/폴더 우클릭 → "공유"
  2. "일반 액세스" → "링크가 있는 모든 사용자"로 변경
  3. 권한 설정
  4. 링크 복사 → 원하는 사람에게 전달
```

> **⚠️ 주의**: "링크가 있는 모든 사용자"로 공유하면 링크를 가진 누구나 접근할 수 있습니다. 민감한 자료는 반드시 특정 사용자에게만 공유하세요.

### 7-3. 공유 드라이브 vs 내 드라이브

| 구분 | 내 드라이브 | 공유 드라이브 |
|------|-----------|-------------|
| 소유자 | 개인 | 팀/조직 |
| 사용 대상 | 개인 파일 관리 | 팀 공동 파일 관리 |
| 멤버 탈퇴 시 | 파일은 본인 소유로 유지 | 파일은 팀에 그대로 남음 |
| 사용 가능 | 모든 구글 계정 | Google Workspace 계정만 |

### 7-4. 용량 관리 (15GB 무료)

무료 계정은 Gmail, Drive, Photos를 합쳐 15GB가 제공됩니다.

**용량 확인**:

```
1. drive.google.com 좌측 하단의 용량 바 확인
   또는
2. drive.google.com/settings → 저장용량 섹션

용량을 많이 차지하는 것들:
  - Gmail 첨부파일 (특히 큰 파일)
  - Drive에 업로드한 파일 (동영상, 사진)
  - Google Photos 원본 크기 사진

용량을 차지하지 않는 것들:
  - Google Docs/Sheets/Slides 자체 (구글 형식)
  - 다른 사람이 나에게 공유한 파일
```

**용량 확보 방법**:

```
1. Drive → 저장용량 (좌측 메뉴) 클릭
2. "사용한 저장용량" 기준으로 정렬
3. 큰 파일부터 확인하여 불필요한 파일 삭제
4. 휴지통 비우기 (휴지통 파일도 용량 차지)
5. Gmail에서 큰 첨부파일이 포함된 메일 삭제
6. Google Photos에서 원본 크기를 "저장 용량 절약" 화질로 변환
```

### 7-5. 오프라인 설정

인터넷이 없는 환경에서도 파일을 볼 수 있도록 설정할 수 있습니다.

```
오프라인 설정 방법 (Chrome 브라우저 필수):
1. drive.google.com → 설정(톱니바퀴)
2. "오프라인" 섹션
3. "오프라인에서 최근 Google 문서, 스프레드시트, 프레젠테이션을
   생성, 열기 및 수정할 수 있습니다." 체크
4. Google 문서 오프라인 확장 프로그램 설치 (안내에 따라)

개별 파일 오프라인 설정:
1. 파일 우클릭 → "오프라인으로 사용 가능" 체크
2. 파일에 ✓ 표시가 나타나면 오프라인 사용 가능
```

### 7-6. 검색 기능 활용

고급 검색 (검색창 우측 필터 아이콘):
  - 파일 유형: 문서, 스프레드시트, PDF 등

  - 수정 날짜: 오늘, 어제, 최근 7일, 최근 30일, 맞춤 기간
  - 파일 이름: 정확한 파일 이름으로 검색

검색 연산자 (파워 유저용):
  type:document         → 문서만
  type:spreadsheet      → 스프레드시트만

  before:2026-01-01     → 특정 날짜 이전

## 8. NotebookLM 입문

### 학습 목표

- NotebookLM의 개념과 기본 사용법을 이해한다
- 소스를 업로드하고 AI에게 질문하여 정보를 추출할 수 있다
- Audio Overview(오디오 브리핑)를 만들 수 있다

### 8-1. NotebookLM이란?

NotebookLM은 Google이 만든 AI 기반 리서치 어시스턴트입니다. 내가 올린 자료(PDF, 웹페이지, 유튜브 영상, Google Docs 등)를 바탕으로 AI가 요약하고, 질문에 답하고, 새로운 콘텐츠를 만들어줍니다.

| 항목 | 설명 |
|------|------|
| 접속 주소 | [notebooklm.google.com](https://notebooklm.google.com/) |
| 비용 | 무료 (Google 계정 필요) |
| 핵심 기능 | AI 요약, Q&A, Audio Overview, 노트 정리 |
| 특징 | 내가 올린 자료만을 근거로 답변 (환각 최소화) |
| 언어 | 한국어 지원 |

**일반 챗봇 vs NotebookLM 차이**:

| 구분 | 일반 AI 챗봇 (ChatGPT, Gemini) | NotebookLM |
|------|-------------------------------|------------|
| 정보 소스 | 인터넷 전체 학습 데이터 | 내가 올린 자료만 |
| 정확도 | 때때로 잘못된 정보 생성 (환각) | 소스 기반이라 환각 최소화 |
| 출처 표시 | 없거나 불분명 | 답변마다 소스 출처 표시 |
| 활용 | 범용 질문, 창의적 작업 | 특정 자료 분석, 연구 |

### 8-2. 소스 업로드

NotebookLM에 업로드할 수 있는 소스 유형입니다.

| 소스 유형 | 설명 | 제한 |
|----------|------|------|
| **PDF** | 문서, 논문, 보고서 | 최대 500,000단어 |
| **Google Docs** | 내 Docs 문서 연결 | Drive에서 선택 |
| **Google Slides** | 내 Slides 연결 | Drive에서 선택 |
| **웹페이지 URL** | 웹 기사, 블로그 | URL 붙여넣기 |
| **YouTube 영상** | 자막이 있는 영상 | URL 붙여넣기 |
| **텍스트 복사/붙여넣기** | 직접 텍스트 입력 | 복사해서 붙여넣기 |
| **Google Drive 오디오** | 음성 파일 | Drive에서 선택 |

**소스 추가 방법**:

```
1. notebooklm.google.com 접속
2. "새 노트북" 클릭
3. 소스 추가:
   - "소스 추가" 버튼 클릭
   - 유형 선택 (Drive, PDF, 웹사이트, YouTube 등)
   - 파일 업로드 또는 URL 입력
4. 하나의 노트북에 여러 소스를 추가할 수 있음 (최대 50개)
```

### 8-3. AI 요약과 Q&A 기능

소스를 업로드하면 NotebookLM이 자동으로 분석하고, 다양한 방식으로 활용할 수 있습니다.

**자동 생성 콘텐츠**:

```
소스를 업로드하면 자동으로:
  - 소스 요약 (핵심 내용 정리)
  - 주요 주제 목록
  - 제안 질문 (AI가 추천하는 질문 목록)
```

**AI에게 질문하기**:

```
질문 예시:
  - "이 문서의 핵심 주장은 무엇인가요?"
  - "저자가 제시한 근거를 정리해주세요"
  - "이 자료에서 언급된 통계를 모두 찾아주세요"
  - "이 내용을 초등학생도 이해할 수 있게 설명해주세요"
  - "자료 A와 자료 B의 공통점과 차이점은?"

답변 특징:
  - 각 답변에 소스 출처가 번호로 표시됨 [1], [2]
  - 출처를 클릭하면 원문의 해당 부분으로 이동
  - 소스에 없는 내용은 "이 소스에서는 관련 정보를 찾을 수 없습니다"라고 답변
```

### 8-4. Audio Overview (오디오 브리핑) 만들기

NotebookLM의 가장 인기 있는 기능입니다. 업로드한 자료를 바탕으로 두 명의 진행자가 대화하는 형태의 오디오 브리핑을 자동 생성합니다.

```
Audio Overview 생성 방법:
1. 소스를 추가한 노트북에서
2. "Audio Overview" 섹션 찾기
3. "생성" 버튼 클릭
4. 커스텀 지시 추가 가능:
   - "한국어로 설명해주세요"
   - "초보자를 대상으로 쉽게 설명해주세요"
   - "특히 ○○ 부분을 자세히 다뤄주세요"
5. 생성 시간: 약 2~5분
6. 재생 또는 다운로드 가능
```

> **팁**: Audio Overview는 영어가 기본이지만, 커스텀 지시에 "한국어로 만들어주세요"라고 입력하면 한국어로 생성됩니다. 통학 시간이나 운동할 때 들으면 효율적으로 학습할 수 있습니다.

### 8-5. 노트 작성과 정리

NotebookLM에서 AI 답변을 노트로 저장하고 정리할 수 있습니다.

```
노트 저장:
1. AI 답변 하단의 "노트에 저장" 클릭
2. 노트 제목 편집
3. 내용 추가/수정 가능

노트 정리:
  - 노트끼리 병합 가능
  - 태그/라벨로 분류
  - 노트를 바탕으로 추가 질문 가능
  - 노트를 Google Docs로 내보내기
```

### 8-6. 활용 사례

| 활용 분야 | 설명 | 소스 예시 |
|----------|------|----------|
| **논문 분석** | 여러 논문을 올려 비교 분석 | PDF 논문 5~10편 |
| **회의록 정리** | 회의 녹음/문서를 요약 | 회의 녹음 파일, 메모 |
| **수업 준비** | 교과서·참고 자료 분석 | 교과서 PDF, 교육과정 문서 |
| **독서 노트** | 책 내용 분석과 정리 | 책 내용 텍스트, 서평 |
| **뉴스 분석** | 여러 기사를 종합 분석 | 웹페이지 URL 여러 개 |
| **영상 학습** | YouTube 강의를 정리 | YouTube URL |

### 실습 과제: PDF 업로드 후 AI 질문

```
[실습] NotebookLM으로 자료 분석하기

1. notebooklm.google.com 접속
2. "새 노트북" 생성
3. 소스 추가:
   - 관심 있는 주제의 웹 기사 URL 2~3개 입력
   (예: "AI 교육 활용", "디지털 리터러시" 관련 기사)

4. AI에게 질문하기 (최소 3가지):
   - "이 자료들의 공통 주장은 무엇인가요?"
   - "핵심 키워드를 5개 뽑아주세요"
   - "이 내용을 3줄로 요약해주세요"

5. 마음에 드는 답변을 노트로 저장 (2개 이상)

6. Audio Overview 생성:
   - 커스텀 지시: "한국어로, 초보자도 이해할 수 있게 설명해주세요"

⏱️ 예상 소요: 20~25분

## 9. Gemini 활용 기초

### 학습 목표

- Gemini의 개념과 모델 종류를 이해한다
- 기본적인 텍스트 대화와 이미지 분석을 할 수 있다
- Google 서비스(Gmail, Docs, Sheets)에서 Gemini를 활용할 수 있다

### 9-1. Gemini란?

Gemini는 Google이 개발한 AI 챗봇입니다. ChatGPT와 비슷하지만, 구글 서비스와 긴밀하게 통합되어 있는 것이 가장 큰 장점입니다.

**접속 방법**:

```
웹: gemini.google.com
앱: Google Gemini 앱 (Android/iOS)
통합: Gmail, Docs, Sheets 등에서 사이드 패널로 접근
```

### 9-2. Gemini 모델 종류

| 모델 | 특징 | 사용 가능 |
|------|------|----------|
| **Gemini Flash** | 빠르고 가벼운 모델, 일상 업무용 | 무료 |
| **Gemini Pro** | 균형 잡힌 성능, 복잡한 작업 | 무료/유료 |
| **Gemini Ultra** | 최고 성능, 고급 분석·창작 | Google One AI Premium |
| **Gemini Nano** | 온디바이스 모델, 오프라인 사용 | Pixel 등 지원 기기 |

> **팁**: 무료 계정으로도 Gemini Flash와 기본 Pro 모델을 사용할 수 있습니다. 일상적인 질문, 글쓰기, 분석에는 무료 모델로 충분합니다.

### 9-3. 기본 사용법

**텍스트 대화**:

```
질문 예시:
  - "파이썬이 뭔지 초등학생도 이해하게 설명해줘"
  - "이메일 답장을 공손하게 써줘: [원문 붙여넣기]"
  - "3월 서울 여행 2박 3일 일정표 짜줘"
  - "SWOT 분석 방법을 알려주고 예시를 보여줘"
  - "이 영어 문장을 자연스러운 한국어로 번역해줘"

효과적인 질문 팁:
  - 구체적으로 요청: "좋은 글 써줘" ✗ → "초등 5학년 대상 독서 감상문 써줘" ✓
  - 형식 지정: "표로 정리해줘", "목록으로 만들어줘"
  - 역할 부여: "초등학교 교사의 관점에서 설명해줘"
  - 예시 제공: "아래와 같은 형식으로 만들어줘: [예시]"
```

**이미지 분석**:

```
이미지 업로드 방법:
1. 채팅창 하단의 이미지 아이콘 (📷) 클릭
2. 이미지 업로드
3. 질문 입력

활용 예시:
  - 식물 사진 → "이 식물의 이름과 관리법을 알려줘"
  - 메뉴판 사진 → "이 메뉴를 한국어로 번역해줘"
  - 수학 문제 사진 → "이 문제 풀이를 설명해줘"
  - 인포그래픽 → "이 이미지의 내용을 텍스트로 정리해줘"
  - 에러 화면 → "이 에러의 원인과 해결 방법을 알려줘"
```

### 9-4. Google 검색 연동 (Grounding)

Gemini는 Google 검색 결과를 실시간으로 참고하여 최신 정보를 반영할 수 있습니다.

```
검색 연동 활용:
  - "오늘 서울 날씨 알려줘" → 실시간 날씨 정보
  - "최근 AI 교육 관련 뉴스 요약해줘" → 최신 기사 기반 답변
  - "○○ 맛집 추천해줘" → Google Maps/리뷰 기반 추천

답변에 출처 링크가 함께 표시되어 원본 확인 가능
```

### 9-5. Google 서비스에서 Gemini 사용

Gemini는 Gmail, Docs, Sheets 등에서 사이드 패널로 바로 접근할 수 있습니다.

**Gmail에서**:

```
- 이메일 요약: 긴 스레드를 빠르게 파악
- 답장 초안: "공손하게 거절하는 답장 작성해줘"
- 이메일 작성: "다음 내용으로 이메일을 작성해줘: ..."
```

**Docs에서**:

```
- 글 작성 도움: "이 주제로 보고서 초안을 작성해줘"
- 요약: "이 문서의 핵심을 3줄로 요약해줘"
- 교정: "이 글의 맞춤법과 문법을 검토해줘"
- 톤 변경: "이 글을 더 공식적인 톤으로 바꿔줘"
```

**Sheets에서**:

```
- 데이터 분석: "이 데이터에서 패턴을 찾아줘"
- 수식 추천: "A열에서 중복을 제거하는 수식을 알려줘"
- 차트 추천: "이 데이터를 시각화하기 좋은 차트 유형을 추천해줘"
```

> **팁**: Docs나 Sheets에서 Gemini를 사용하면 현재 문서의 내용을 맥락으로 이해하기 때문에 더 정확한 답변을 받을 수 있습니다.

### 실습 과제: Gemini로 여행 계획 세우기

## 10. Google Maps·번역·렌즈

### 학습 목표

- Google Maps로 길찾기, 장소 저장, 리뷰 확인을 할 수 있다
- Google 번역으로 텍스트, 음성, 카메라 번역을 할 수 있다
- Google 렌즈로 이미지 검색과 텍스트 인식을 할 수 있다

### 10-1. Google Maps

Google Maps는 단순한 지도 앱이 아니라 길찾기, 장소 탐색, 리뷰 확인까지 가능한 종합 위치 서비스입니다.

**핵심 기능**:

| 기능 | 설명 | 사용법 |
|------|------|--------|
| **길찾기** | 자동차, 대중교통, 도보, 자전거 경로 | 출발지→도착지 입력 |
| **장소 저장** | 즐겨찾는 장소를 저장 | 장소 클릭 → ♡ 저장 |
| **오프라인 지도** | 인터넷 없이 지도 사용 | 영역 선택 → 다운로드 |
| **타임라인** | 내 이동 기록 확인 | 프로필 → 타임라인 |
| **리뷰** | 다른 사용자의 평가 확인 | 장소 클릭 → 리뷰 탭 |
| **스트리트 뷰** | 360도 거리 사진 | 사람 모양 아이콘 드래그 |
| **실시간 교통** | 현재 교통 상황 확인 | 레이어 → 교통정보 |

**유용한 팁**:

```
- 여러 경유지 추가: 길찾기에서 "경유지 추가" 클릭
- 나만의 지도 만들기: maps.google.com/mymaps → 여행 계획, 맛집 리스트 등
- 공유: 장소나 경로를 링크로 공유 가능
- 리뷰 작성: 방문한 장소에 사진과 후기 남기기
```

### 10-2. Google 번역

Google 번역은 133개 이상의 언어를 지원하는 번역 서비스입니다.

**번역 방식**:

| 방식 | 설명 | 사용법 |
|------|------|--------|
| **텍스트 번역** | 입력한 텍스트 번역 | translate.google.com에서 입력 |
| **문서 번역** | 문서 파일 전체 번역 | "문서" 탭 → 파일 업로드 |
| **웹사이트 번역** | 웹페이지 전체 번역 | "웹사이트" 탭 → URL 입력 |
| **음성 번역** | 말하면 실시간 번역 | 마이크 아이콘 → 말하기 |
| **카메라 번역** | 카메라로 비추면 번역 | 앱 → 카메라 아이콘 |
| **손글씨 번역** | 손으로 글자 그리기 | 펜 아이콘 → 글자 그리기 |
| **대화 번역** | 두 언어 실시간 통역 | 앱 → 대화 모드 |

**실용적 활용**:

```
- 해외여행: 메뉴판, 간판, 안내문 카메라 번역
- 업무: 영어 이메일 번역 후 답장 초안 작성
- 학습: 외국어 발음 확인 (스피커 아이콘)
- 문서: PDF/DOCX 파일 통째로 번역
```

> **팁**: 카메라 번역은 Google 번역 앱에서 사용할 수 있습니다. 카메라를 비추면 원문 위에 번역이 실시간으로 표시됩니다.

### 10-3. Google 렌즈

Google 렌즈는 카메라나 이미지로 정보를 검색하는 시각 검색 도구입니다.

| 기능 | 설명 | 활용 |
|------|------|------|
| **이미지 검색** | 사진으로 유사한 이미지 검색 | 제품명 모를 때, 식물/동물 식별 |
| **텍스트 인식 (OCR)** | 이미지에서 텍스트 추출 | 명함, 문서, 간판의 텍스트 복사 |
| **번역** | 이미지 속 텍스트 번역 | 외국어 메뉴판, 안내문 |
| **제품 검색** | 사진으로 비슷한 제품 찾기 | 쇼핑, 가격 비교 |
| **장소 식별** | 건물/랜드마크 식별 | 관광지, 건물 정보 |
| **숙제 도움** | 수학 문제 풀이 | 수학 문제 사진 촬영 |

**사용 방법**:

```
스마트폰:
  - Google 앱 → 검색창의 렌즈(카메라) 아이콘
  - Google 렌즈 앱 직접 실행
  - 카메라 앱 → Google 렌즈 연동

PC:
  - Google 이미지 검색 → 카메라 아이콘
  - 이미지 파일 업로드 또는 URL 입력
  - Chrome 브라우저에서 이미지 우클릭 → "Google 렌즈로 검색"
```

### 실습 과제: 일상에서 활용하기

```
[실습] Google Maps·번역·렌즈 활용 체험

1. Google Maps:
   - 집에서 가장 가까운 도서관까지 대중교통 길찾기
   - 도서관을 "저장 목록"에 추가
   - 도서관 리뷰 확인

2. Google 번역:
   - translate.google.com 접속
   - 아무 한국어 문장을 영어, 일본어, 중국어로 번역
   - 번역된 텍스트의 스피커 아이콘으로 발음 확인
   - 영어 → 한국어로 짧은 뉴스 기사 문단 번역

3. Google 렌즈:
   - 책 표지 사진을 Google 렌즈로 검색
   - 검색 결과에서 책 정보, 가격 등 확인
   - (스마트폰) 주변 간판이나 제품 라벨을 렌즈로 스캔

⏱️ 예상 소요: 15~20분
```

## 11. 종합 실습

### 학습 목표

- 지금까지 배운 구글 도구를 연계하여 하나의 프로젝트를 완성할 수 있다
- 각 도구의 강점을 파악하고 적절한 상황에 맞는 도구를 선택할 수 있다

### 프로젝트: "구글 도구로 이벤트 기획서 만들기"

지금까지 배운 구글 도구를 모두 활용하여 작은 이벤트(동아리 모임, 학급 행사, 동호회 활동 등)를 기획해봅니다.

### Step 1: Gemini로 아이디어 브레인스토밍

```
Gemini에게 요청:
  "초등학교 3학년 학급에서 진행할 수 있는
  봄맞이 야외 활동 아이디어 5가지를 제안해줘.
  각 아이디어마다 준비물, 소요시간, 난이도를 표로 정리해줘."

결과를 복사해둡니다.
```

### Step 2: Docs로 기획서 초안 작성

```
Google Docs에서 기획서 작성:

구성:
  - 제목: ○○ 봄맞이 야외 활동 기획서
  - 제목 1: 행사 개요 (일시, 장소, 대상, 목적)
  - 제목 1: 프로그램 일정 (시간표 형태)
  - 제목 1: 준비물 목록
  - 제목 1: 역할 분담
  - 제목 1: 예산 계획 (개략)
  - 제목 1: 비상 연락처 및 안전 수칙

Gemini 결과를 참고하여 내용 채우기
목차 자동 생성
```

### Step 3: Sheets로 예산표 작성

```
Google Sheets에서 예산표 만들기:

A열: 항목 | B열: 단가 | C열: 수량 | D열: 소계 | E열: 비고

데이터 입력 (10개 이상 항목)
  - 간식비, 교통비, 재료비, 인쇄비, 기타 등

함수 활용:
  - D열: =B2*C2 (소계 자동 계산)
  - 합계: =SUM(D2:D20)
  - 차트: 항목별 예산 비율 원형 차트
```

### Step 4: Forms로 참가 신청서 만들기

```
Google Forms로 참가 신청서:

문항:
  - 이름 (단답형, 필수)
  - 반 (드롭다운: 1반~6반)
  - 참가 여부 (객관식: 참가/불참가)
  - 알레르기 유무 (객관식 + 기타 입력)
  - 보호자 연락처 (단답형)
  - 하고 싶은 활동 (체크박스, 복수 선택)

스프레드시트 연동 설정
```

### Step 5: Slides로 발표 자료 제작

```
Google Slides로 발표 자료:

5~8장 구성:
  1장: 제목 (행사명, 일시)
  2장: 행사 목적 및 기대 효과
  3장: 프로그램 일정표
  4장: 준비물 및 역할 분담
  5장: 예산 요약 (Sheets 차트 삽입)
  6장: 신청 방법 (Forms QR코드 또는 링크)
  7장: 안전 수칙
  8장: Q&A
```

### Step 6: Drive에 모든 파일 정리

```
Google Drive 폴더 구성:

📁 봄맞이 야외 활동
├── 📄 기획서 (Docs)
├── 📊 예산표 (Sheets)
├── 📋 참가 신청서 (Forms)
├── 📽️ 발표 자료 (Slides)
├── 📁 참고 자료
└── 📁 사진 (행사 후)

폴더를 관련 교사에게 공유 (편집자 권한)
```

### Step 7: NotebookLM으로 관련 자료 분석

```
NotebookLM 활용:

1. 새 노트북 생성: "야외 활동 기획 참고"
2. 소스 추가:
   - 야외 활동 관련 교육 자료 URL 2~3개
   - 작성한 기획서 (Google Docs 연결)
3. AI에게 질문:
   - "이 기획서에서 빠진 부분이 있을까요?"
   - "안전 수칙에 추가할 사항을 제안해주세요"
   - "비 올 경우 대안 활동을 추천해주세요"
4. 유용한 답변을 노트로 저장
```

### 완성 체크리스트

```
✅ 종합 실습 체크리스트:
□ Gemini로 아이디어 브레인스토밍 완료
□ Docs 기획서 작성 (목차 포함)
□ Sheets 예산표 작성 (함수, 차트 포함)
□ Forms 참가 신청서 작성 (5개 이상 문항)
□ Slides 발표 자료 제작 (5장 이상)
□ Drive에 폴더 구성 및 파일 정리
□ NotebookLM으로 기획서 보완
□ 관련 사람에게 폴더 공유

⏱️ 예상 소요: 60~90분
```

## 12. 트러블슈팅 & FAQ

### 자주 발생하는 문제와 해결책

**문제 1: Google Drive 용량이 부족합니다**

```
원인: Gmail + Drive + Photos 합산 15GB 초과
해결:
  1. drive.google.com → 저장용량 클릭
  2. 큰 파일부터 정렬하여 불필요한 파일 삭제
  3. 휴지통 비우기 (휴지통도 용량 차지)
  4. Gmail에서 큰 첨부파일 포함 메일 검색: "has:attachment larger:10M"
  5. Google Photos 화질을 "저장 용량 절약"으로 변환
  6. 필요시 Google One 구독 (100GB: 월 2,400원)
```

**문제 2: 공유 문서를 편집할 수 없습니다**

```
원인: "뷰어" 권한만 부여됨
해결:
  1. 문서 상단의 "편집 액세스 요청" 클릭
  2. 문서 소유자에게 편집 권한 변경 요청
  3. 소유자: 공유 설정에서 "뷰어" → "편집자"로 변경
```

**문제 3: Google Docs에서 글꼴이 깨져 보입니다**

```
원인: Word 파일을 Docs로 변환할 때 호환성 문제
해결:
  1. 원본 Word 파일에서 사용한 글꼴이 Google Docs에 없을 수 있음
  2. 텍스트 선택 → 지원되는 글꼴로 변경
  3. 추천 한국어 글꼴: 나눔고딕, 나눔명조, 맑은 고딕
```

**문제 4: Sheets 함수가 에러를 표시합니다**

```
흔한 에러와 해결:
  #REF!  → 참조 범위 오류 → 범위 다시 확인
  #N/A   → 값을 찾을 수 없음 → VLOOKUP 검색값 확인
  #VALUE! → 값 유형 오류 → 숫자/텍스트 타입 확인
  #DIV/0! → 0으로 나눔 → =IF(B1=0, 0, A1/B1) 사용
  #NAME?  → 함수 이름 오류 → 철자 확인
  #ERROR! → 구문 오류 → 괄호, 쉼표 확인
```

**문제 5: Forms 설문 링크를 받은 사람이 접근할 수 없습니다**

```
원인: 조직 외부 응답이 차단됨 (Workspace 계정인 경우)
해결:
  1. Forms 설정 → "응답" 섹션
  2. "[조직명] 사용자로 제한"을 해제
  3. 또는 개인 Gmail 계정으로 양식을 만들면 누구나 접근 가능
```

**문제 6: NotebookLM에 소스가 추가되지 않습니다**

```
원인: 파일 크기 초과 또는 지원하지 않는 형식
해결:
  1. PDF 크기 확인 (500,000단어 이하)
  2. 지원 형식 확인: PDF, Google Docs/Slides, 웹 URL, YouTube
  3. 스캔한 PDF(이미지 PDF)는 인식되지 않을 수 있음
     → OCR 처리된 PDF 사용
  4. YouTube는 자막이 있는 영상만 지원
```

**문제 7: Gemini가 한국어로 답변하지 않습니다**

```
해결:
  1. 질문을 한국어로 입력 (영어로 물으면 영어로 답함)
  2. 명시적으로 요청: "한국어로 답변해주세요"
  3. 설정에서 기본 언어를 한국어로 변경
  4. gemini.google.com → 설정 → 언어 확인
```

**문제 8: 오프라인에서 Google 문서를 편집할 수 없습니다**

```
원인: 오프라인 모드가 설정되지 않음
해결:
  1. 인터넷이 연결된 상태에서 미리 설정 필요
  2. Chrome 브라우저에서만 지원 (다른 브라우저 불가)
  3. drive.google.com → 설정 → "오프라인" 활성화
  4. 오프라인으로 사용할 파일을 미리 열어두면 캐시됨
  5. 편집 내용은 인터넷 재연결 시 자동 동기화
```

### 자주 묻는 질문 (FAQ)

**Q1: Google Docs와 Microsoft Word, 어떤 걸 써야 하나요?**

> 협업이 많고 클라우드 기반 작업이 필요하면 Google Docs, 복잡한 서식이나 고급 편집이 필요하면 Word를 추천합니다. Docs에서 Word 파일을 열고 편집할 수도 있으므로, 둘 다 사용하는 것도 좋은 방법입니다.

**Q2: 구글 Drive 용량 15GB가 부족하면 어떻게 하나요?**

> 우선 불필요한 파일과 휴지통을 정리하세요. 그래도 부족하면 Google One을 구독할 수 있습니다 (100GB: 월 2,400원, 200GB: 월 3,700원, 2TB: 월 11,900원). 학교 Workspace 계정이라면 더 큰 용량이 제공될 수 있습니다.

**Q3: NotebookLM과 Gemini의 차이점이 뭔가요?**

> Gemini는 범용 AI 챗봇으로 다양한 질문에 답합니다. NotebookLM은 내가 올린 특정 자료만을 근거로 답변하므로 정확도가 높고 출처를 명확히 보여줍니다. 일반 질문은 Gemini, 특정 자료 분석은 NotebookLM을 사용하세요.

**Q4: 구글 서비스는 개인정보가 안전한가요?**

> 구글은 강력한 보안 시스템을 갖추고 있으나, 사용자도 2단계 인증, 강력한 비밀번호 사용, 공유 권한 관리 등에 신경 써야 합니다. 민감한 정보는 공유 범위를 최소화하고, 정기적으로 보안 점검을 실행하세요.

**Q5: 스마트폰에서도 구글 도구를 사용할 수 있나요?**

> 모든 구글 도구는 스마트폰 앱으로 제공됩니다. Play 스토어(Android) 또는 App Store(iOS)에서 Google Docs, Sheets, Slides, Drive, Forms 앱을 설치하세요. PC와 동일한 파일이 자동 동기화됩니다.

**Q6: 여러 구글 계정을 동시에 사용할 수 있나요?**

> 가능합니다. Chrome 브라우저에서 프로필 아이콘을 클릭하면 다른 계정을 추가할 수 있습니다. 또는 Chrome 프로필을 별도로 만들면 계정별로 완전히 분리된 환경을 사용할 수 있습니다.

**Q7: Google Forms 응답 데이터를 엑셀로 내보낼 수 있나요?**

> 가능합니다. Forms의 응답을 스프레드시트에 연동한 후, 해당 스프레드시트에서 파일 → 다운로드 → Microsoft Excel(.xlsx)로 내보내면 됩니다. 또는 CSV 형식으로도 내보낼 수 있습니다.

**Q8: Gemini는 무료인가요?**

> 기본 Gemini (Flash, Pro 모델)는 무료입니다. 고급 기능(Ultra 모델, 100만 토큰 컨텍스트, Gems 등)은 Google One AI Premium (월 $19.99)에서 사용할 수 있습니다. 일반적인 사용에는 무료 모델로 충분합니다.
