1. 구글 생태계란?

1-2. 구글 생태계의 핵심 강점

1. 무료 — 대부분의 핵심 서비스를 무료로 이용할 수 있습니다.

  • Gmail 15GB, Drive 15GB, Docs/Sheets/Slides 무제한 생성
  • 추가 비용 없이 팀 협업 기능까지 제공

2. 통합 — 하나의 계정으로 모든 서비스가 연결됩니다.

  • Gmail에서 받은 첨부파일이 자동으로 Drive에 저장
  • Docs에서 작성한 문서를 Slides에서 바로 참조
  • Calendar 일정에 Meet 링크 자동 생성

3. 클라우드 — 인터넷만 있으면 어디서든 접근할 수 있습니다.

  • 설치할 프로그램이 없습니다 (브라우저에서 모두 실행)
  • 자동 저장 — 별도로 저장 버튼을 누를 필요가 없습니다
  • PC, 스마트폰, 태블릿 어디서든 동일한 파일에 접근

4. 협업 — 여러 사람이 동시에 같은 문서를 편집할 수 있습니다.

  • 실시간 공동 편집 (다른 사람의 커서가 보임)
  • 댓글, 제안 모드로 피드백 교환
  • 공유 링크 하나로 누구에게나 전달 가능

2. Google 계정과 Workspace

학습 목표

  • 구글 계정을 만들고 기본 설정을 관리할 수 있다
  • 2단계 인증을 설정하여 계정 보안을 강화할 수 있다
  • 계정 복구 방법을 알고 대비할 수 있다

2-1. 구글 계정 만들기

구글 계정은 모든 구글 서비스의 출발점입니다. 이미 Gmail 주소가 있다면 별도 가입 없이 바로 사용할 수 있습니다.

1. accounts.google.com 접속
2. "계정 만들기" 클릭
3. 이름 입력 (성, 이름)
4. 원하는 Gmail 주소 선택 (예: myname@gmail.com)
5. 비밀번호 설정 (8자 이상, 영문+숫자+특수문자 조합 권장)
6. 전화번호 인증 (선택이지만 보안을 위해 권장)
7. 복구 이메일 설정 (비밀번호 분실 시 필요)
8. 생년월일, 성별 입력
9. 약관 동의 후 완료

주의: 이미 구글 계정이 있다면 새로 만들 필요 없습니다. 하나의 계정으로 모든 서비스를 이용할 수 있습니다.

2-2. 계정 관리하기

계정 설정은 myaccount.google.com에서 관리합니다.

2-3. 2단계 인증 설정 (필수!)

2단계 인증은 비밀번호가 유출되더라도 계정을 보호하는 가장 중요한 보안 설정입니다.

1. myaccount.google.com → 보안 탭
2. "Google에 로그인하는 방법" 섹션
3. "2단계 인증" 클릭 → "시작하기"
4. 인증 방법 선택:
   - Google 메시지 (추천): 스마트폰에 알림
   - 인증 앱: Google Authenticator 등
   - 문자 메시지: SMS 인증 코드
   - 보안 키: 물리적 USB 키
5. 백업 코드 저장 (반드시 안전한 곳에 보관!)

⚠️ 중요: 백업 코드는 스마트폰을 분실했을 때 계정에 접근하는 유일한 방법입니다. 반드시 인쇄하거나 안전한 곳에 저장하세요.

3. Google Docs

학습 목표

  • Google Docs에서 새 문서를 만들고 기본 서식을 적용할 수 있다
  • 실시간 협업 기능(공유, 댓글, 제안 모드)을 사용할 수 있다
  • 문서를 다양한 형식(PDF, Word)으로 내보낼 수 있다

3-1. 문서 작성 기초

Google Docs는 클라우드 기반 문서 작성 도구입니다. Microsoft Word와 비슷하지만, 설치 없이 브라우저에서 바로 사용할 수 있고, 자동 저장됩니다.

새 문서 만들기:
  방법 1: docs.google.com → "빈 문서" 클릭
  방법 2: drive.google.com → "새로 만들기" → "Google 문서"
  방법 3: 브라우저 주소창에 docs.new 입력

팁: docs.new를 주소창에 입력하면 즉시 새 문서가 열립니다. sheets.new, slides.new, forms.new도 마찬가지입니다!

3-3. 제목/본문 스타일과 목차

스타일 적용:
  1. 텍스트 선택 → "일반 텍스트" 드롭다운 클릭
  2. 스타일 선택: 제목/부제목/제목 1~3/일반 텍스트

목차 자동 생성:
  1. 커서를 원하는 위치에 놓기
  2. 메뉴 → 삽입 → 목차 → 스타일 선택
  3. 제목 변경 시 목차 옆 🔄 버튼으로 업데이트

팁: 제목 스타일을 쓰면 문서 좌측 "개요" 패널에서도 빠르게 이동할 수 있습니다. (보기 → 개요 표시)

3-4. 실시간 협업

문서 공유하기:
  1. 우측 상단 "공유" 버튼 클릭
  2. 특정 사용자: 이메일 입력 / 링크 공유: "링크가 있는 모든 사용자"
  3. 권한: 뷰어(보기만) / 댓글 작성자 / 편집자

댓글 달기:
  텍스트 선택 → 💬 아이콘 (또는 Ctrl+Alt+M) → 댓글 입력

제안 모드:
  우측 상단 ✏️ → "제안" 선택 → 원본 유지하며 수정 제안

팁: 제안 모드는 교사가 학생 글을 첨삭하거나, 팀원이 보고서를 검토할 때 매우 유용합니다.

3-5. 음성 입력 활용

1. 메뉴 → 도구 → 음성 입력 (또는 Ctrl + Shift + S)
2. 마이크 아이콘 클릭 → 말하기 → 자동 텍스트 변환
음성 명령: "마침표" → .  "쉼표" → ,  "줄 바꿈" "새 단락"

주의: 음성 입력은 Chrome 브라우저에서만 지원됩니다.

3-6. 문서 내보내기

메뉴 → 파일 → 다운로드
지원: Word(.docx), PDF, RTF, TXT, HTML, EPUB

실습 과제: 간단한 보고서 작성

[실습] "내가 좋아하는 여행지" 보고서 만들기

1. docs.new로 새 문서 생성
2. 제목: "내가 좋아하는 여행지 TOP 3"
3. 제목 스타일 적용 (제목 1: 여행지, 제목 2: 위치/추천이유/꿀팁)
4. 각 여행지 2~3줄 설명 + 이미지 삽입
5. 목차 자동 생성 → PDF 내보내기

⏱️ 예상 소요: 20~30분

4. Google Sheets

학습 목표

  • Google Sheets의 기본 인터페이스와 셀/행/열 개념을 이해한다
  • 핵심 함수 10가지를 사용하여 데이터를 계산할 수 있다
  • 차트를 만들고, 필터와 정렬로 데이터를 정리할 수 있다

4-1. 스프레드시트 기초

Google Sheets는 데이터를 표 형태로 관리하고 계산하는 도구입니다. Microsoft Excel과 유사합니다.

용어설명예시
셀(Cell)데이터를 입력하는 가장 작은 단위A1, B3, C10
행(Row)가로줄 (숫자로 표시)1행, 2행, 3행...
열(Column)세로줄 (알파벳으로 표시)A열, B열, C열...
시트(Sheet)하나의 탭 (하단에 탭으로 표시)Sheet1, Sheet2
범위(Range)여러 셀의 묶음A1:C10
수식(Formula)계산을 위한 식 (= 으로 시작)=A1+B1

  방법 3: Drive → 새로 만들기 → Google 스프레드시트

4-2. 핵심 함수 10선

함수는 =함수이름(인수) 형태로 셀에 입력합니다.

1. SUM — 합계
  =SUM(A1:A10)          → A1~A10의 합계

2. AVERAGE — 평균
  =AVERAGE(A1:A10)      → A1~A10의 평균값

3. COUNT / COUNTA — 개수 세기
  =COUNT(A1:A10)        → 숫자가 있는 셀 개수
  =COUNTA(A1:A10)       → 비어있지 않은 셀 개수

4. MAX / MIN — 최대값 / 최소값
  =MAX(A1:A10)          → 최대값
  =MIN(A1:A10)          → 최소값

5. IF — 조건 판단
  =IF(A1>=60, "합격", "불합격")

6. VLOOKUP — 세로 검색
  =VLOOKUP("사과", A1:C10, 3, FALSE)

7. CONCATENATE — 텍스트 합치기
  =A1 & " " & B1

8. TODAY / NOW — 오늘 날짜 / 현재 시간
  =TODAY()    =NOW()

9. COUNTIF — 조건부 개수
  =COUNTIF(A1:A10, "합격")

10. SUMIF — 조건부 합계
  =SUMIF(A1:A10, "서울", C1:C10)

팁: 함수를 외울 필요 없습니다! 셀에 =을 입력하고 함수 이름의 처음 몇 글자만 치면 자동완성 목록이 나타납니다.

12. 트러블슈팅 & FAQ

자주 발생하는 문제와 해결책

문제 1: Google Drive 용량이 부족합니다

원인: Gmail + Drive + Photos 합산 15GB 초과
해결: 큰 파일 정렬 삭제, 휴지통 비우기,
  Gmail "has:attachment larger:10M" 검색,
  Photos 화질 변환, 필요시 Google One 구독

문제 2: 공유 문서를 편집할 수 없습니다

원인: "뷰어" 권한만 부여됨
해결: "편집 액세스 요청" 클릭 → 소유자가 편집자로 변경

문제 3: Google Docs에서 글꼴이 깨져 보입니다

원인: Word → Docs 변환 시 호환성 문제
해결: 지원되는 글꼴로 변경 (나눔고딕, 나눔명조, 맑은 고딕)

문제 4: Sheets 함수 에러

#REF!   → 참조 범위 오류
#N/A    → 값을 찾을 수 없음
#VALUE! → 값 유형 오류
#DIV/0! → 0으로 나눔 → =IF(B1=0, 0, A1/B1) 사용
#NAME?  → 함수 이름 철자 확인
#ERROR! → 괄호, 쉼표 확인

문제 5: Forms 설문 접근 불가

원인: Workspace 계정에서 조직 외부 응답 차단
해결: Forms 설정 → "사용자로 제한" 해제 또는 개인 Gmail로 작성

문제 6: NotebookLM 소스 추가 불가

해결: PDF 크기 확인 (500,000단어 이하),
  지원 형식 확인, 스캔 PDF는 OCR 처리 필요,
  YouTube는 자막이 있는 영상만 지원

문제 7: Gemini가 한국어로 답변하지 않음

해결: 한국어로 질문, "한국어로 답변해주세요" 명시,
  설정에서 기본 언어를 한국어로 변경

문제 8: 오프라인에서 편집 불가

해결: 인터넷 연결 시 미리 설정 필요, Chrome만 지원,
  Drive 설정 → "오프라인" 활성화, 파일 미리 열어두기

자주 묻는 질문 (FAQ)

Q1: Google Docs와 Microsoft Word, 어떤 걸 써야 하나요?

협업이 많고 클라우드 기반 작업이 필요하면 Google Docs, 복잡한 서식이나 고급 편집이 필요하면 Word를 추천합니다. Docs에서 Word 파일을 열고 편집할 수도 있습니다.

Q2: Drive 용량 15GB가 부족하면?

불필요한 파일과 휴지통을 정리하세요. 그래도 부족하면 Google One을 구독할 수 있습니다 (100GB: 월 2,400원). 학교 Workspace 계정이라면 더 큰 용량이 제공될 수 있습니다.

Q3: NotebookLM과 Gemini의 차이점?

Gemini는 범용 AI 챗봇, NotebookLM은 내가 올린 특정 자료만을 근거로 답변합니다. 일반 질문은 Gemini, 특정 자료 분석은 NotebookLM을 사용하세요.

Q4: 구글 서비스 개인정보 안전한가요?

구글은 강력한 보안 시스템을 갖추고 있으나, 사용자도 2단계 인증, 강력한 비밀번호, 공유 권한 관리에 신경 써야 합니다.

Q5: 스마트폰에서도 사용 가능한가요?

모든 구글 도구는 스마트폰 앱으로 제공됩니다. Play 스토어(Android) 또는 App Store(iOS)에서 설치하세요.

Q6: 여러 구글 계정을 동시에 사용할 수 있나요?

가능합니다. Chrome에서 프로필 아이콘 클릭 → 다른 계정 추가. 또는 Chrome 프로필을 별도로 만들면 완전히 분리된 환경을 사용할 수 있습니다.

Q7: Forms 응답을 엑셀로 내보낼 수 있나요?

가능합니다. 응답을 스프레드시트에 연동한 후 파일 → 다운로드 → Excel(.xlsx) 또는 CSV로 내보내면 됩니다.

Q8: Gemini는 무료인가요?

기본 Gemini (Flash, Pro)는 무료입니다. 고급 기능(Ultra 모델)은 Google One AI Premium (월 $19.99)에서 사용할 수 있습니다.
