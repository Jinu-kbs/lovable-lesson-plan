## 2. 목적 및 배경

### 배경

2026년 3월 "She Builds on Lovable" 이벤트를 계기로 시작된 프로젝트로, 바이브코딩(Vibe Coding) 도구별 맞춤 교안을 제공한다. 러버블(Lovable), 안티그래비티(AntiGravity), 커서(Cursor), AI Studio Builder, Claude Code 등 다양한 바이브코딩 도구를 포괄하는 멀티 도구 가이드 사이트이다.

### 목적

- 여러 바이브코딩 도구에 대한 수준별(초보/중급/개발자) 맞춤 교안 제공
- 도구별 특징 비교 및 사용자 수준에 맞는 도구 추천
- 사전 준비부터 배포까지 전 과정을 한 곳에서 안내
- 이벤트 혜택(25시간 무료 빌드, $100 API 크레딧, $250 Stripe 크레딧) 활용 가이드 포함

## 3. 대상 사용자

| 수준 | 대상 | 특징 |
|------|------|------|
| **초보자** | AI 챗봇 경험 없는 완전 입문자 | 용어부터 설명, 단계별 스크린샷 필요 |
| **중급자** | AI 도구(ChatGPT, 제미나이 등) 경험자 | 기획 문서(PRD/IA/ERD) 작성법, 도구 비교 필요 |
| **개발자** | 코딩 기초(HTML/JS/Python 등) 보유자 | 기술 아키텍처 이해, 코드 커스터마이징, 마이그레이션 필요 |

### 자가 진단 흐름 (인터랙티브 퀴즈)

```
Q1: 어떤 것을 만들고 싶으신가요?
    웹사이트·웹앱 → 러버블
    모바일 앱 → 안티그래비티
    코드 직접 제어 → 커서
    빠른 프로토타입 → AI Studio (초보자만)
    터미널 기반 개발 → Claude Code (개발자만)

Q2: 현재 기술 수준은? (단일 수준 도구는 Q2 생략)
    입문자 → 초보자 교안
    중급자 → 중급자 교안
    개발 경험자 → 개발자 교안

→ 추천 결과: 도구 + 수준 조합 교안 링크 제공
```

## 8. 개인정보 보호 요구사항

- 카카오톡 대화에서 발췌한 인용문은 반드시 **익명 처리**
  - 허용: `— 실사용자 후기`, `— 실사용자 조언`, `— 실사용자 경험`
  - 금지: 실명, 닉네임 등 개인 식별 정보 사용
- 리소스/링크의 제작자 크레딧 표기는 **허용**
  - 허용: `(peace 제작/공유)`, `(이운희 제작)`
- 카카오톡 원본 파일(`.txt`)은 **커밋하지 않음** (개인정보 포함)
- PDF 참고자료는 **커밋하지 않음** (용량)
