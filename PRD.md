## 2. 목적 및 배경

### 배경

2026년 3월 "She Builds on Lovable" 이벤트가 개최되며, 참가자들에게 수준별 맞춤 교안이 필요하다. 바이브코딩(Vibe Coding)이라는 새로운 개발 방식이 주목받고 있으며, 러버블(Lovable)은 대표적인 AI 기반 웹 앱 빌더이다. 비개발자부터 개발자까지 다양한 수준의 참가자가 러버블을 활용하여 웹사이트를 만들 수 있도록 단계별 교안을 제공한다.

### 목적

- 바이브코딩과 러버블에 대한 수준별(초보/중급/개발자) 입문 교안 제공
- 사전 준비부터 배포까지 전 과정을 한 곳에서 안내
- 이벤트 혜택(25시간 무료 빌드, $100 API 크레딧, $250 Stripe 크레딧) 활용 가이드 포함

## 3. 대상 사용자

| 수준 | 대상 | 특징 |
|------|------|------|
| **초보자** | AI 챗봇 경험 없는 완전 입문자 | 용어부터 설명, 단계별 스크린샷 필요 |
| **중급자** | AI 도구(ChatGPT, 제미나이 등) 경험자 | 기획 문서(PRD/IA/ERD) 작성법, 도구 비교 필요 |
| **개발자** | 코딩 기초(HTML/JS/Python 등) 보유자 | 기술 아키텍처 이해, 코드 커스터마이징, 마이그레이션 필요 |

### 자가 진단 흐름

```
Q1: AI 챗봇 경험? ─ 아니오 → 초보자편
                     예 → Q2

Q2: 개발 용어 이해? ─ 아니오 → 중급자편 (초보자편도 참고)
                      예 → Q3

Q3: 코딩 경험? ─ 아니오 → 중급자편
                 예 → 개발자편
```

## 8. 개인정보 보호 요구사항

- 카카오톡 대화에서 발췌한 인용문은 반드시 **익명 처리**
  - 허용: `— 실사용자 후기`, `— 실사용자 조언`, `— 실사용자 경험`
  - 금지: 실명, 닉네임 등 개인 식별 정보 사용
- 리소스/링크의 제작자 크레딧 표기는 **허용**
  - 허용: `(peace 제작/공유)`, `(이운희 제작)`
- 카카오톡 원본 파일(`.txt`)은 **커밋하지 않음** (개인정보 포함)
- PDF 참고자료는 **커밋하지 않음** (용량)
