2. 도구별 상세

💗 러버블 (Lovable)

웹 브라우저에서 자연어로 지시하면 풀스택 웹앱을 자동으로 만들어주는 노코드 빌더입니다. React, Tailwind CSS, Supabase 등 현대적 기술 스택을 자동 구성합니다.

✅ 장점
  • 설치 없이 브라우저에서 바로 시작
  • 코딩 없이 풀스택 웹앱 제작
  • 원클릭 배포 (Publish)
  • Supabase DB, Stripe 결제 연동
  • She Builds 이벤트 혜택 (25시간 무료)
⚠️ 단점
  • 웹사이트/웹앱에 한정
  • 무료 후 유료 전환 필요
  • 세밀한 코드 제어 어려움
  • 영어 UI
🌱 초보자편 🚀 중급자편 ⚙️ 개발자편

🚀 안티그래비티 (AntiGravity)

Google이 만든 에이전트 기반 IDE입니다. VS Code를 기반으로 AI 에이전트가 자율적으로 코딩·테스트·브라우저 조작까지 수행합니다. Google 생태계와 자연스럽게 통합됩니다.

✅ 장점
  • 에이전트가 자율적으로 작업 수행
  • Google 생태계 통합 (Firebase, Cloud)
  • 병렬 에이전트로 동시 작업
  • 퍼블릭 프리뷰 무료
  • 웹·모바일·데스크톱 모든 프로젝트
⚠️ 단점
  • 데스크톱 앱 설치 필요
  • Google 계정 필수
  • 아직 프리뷰 단계
  • 영어 UI
🌱 초보자편 🚀 중급자편 ⚙️ 개발자편

💻 커서 (Cursor)

VS Code 기반의 AI 코드 에디터입니다. Tab(자동완성), Composer(멀티파일 편집), Agent(자율 실행) 3단계 AI로 코드를 직접 제어하며 개발합니다. VS Code 확장 기능 생태계와 호환됩니다.

✅ 장점
  • VS Code 생태계 완벽 호환
  • Tab·Composer·Agent 3단계 AI
  • 다중 AI 모델 선택 가능
  • Hobby 플랜 무료
  • 코드 직접 제어 가능
⚠️ 단점
  • 데스크톱 앱 설치 필요
  • 코딩 기초 이해 필요
  • 무료 플랜 제한적
  • 자체 배포 기능 없음
🌱 초보자편 🚀 중급자편 ⚙️ 개발자편

🧪 AI Studio (Google)

Google AI Studio의 Build 탭에서 Gemini 2.5 Pro와 대화하며 웹 프로토타입을 만드는 가장 간단한 도구입니다. Google 계정만 있으면 설치 없이 바로 시작할 수 있습니다.

✅ 장점
  • 가장 쉽고 빠른 시작
  • 설치 불필요 (웹 브라우저)
  • Google 계정으로 무료 사용
  • 빠른 프로토타입 제작
⚠️ 단점
  • 간단한 프로토타입에 한정
  • 복잡한 백엔드 불가
  • 제한적 배포 옵션
  • 데이터베이스 연동 어려움
🌱 초보자편

⚡ Claude Code (Anthropic)

터미널에서 Claude와 대화하며 코딩하는 에이전트형 CLI 도구입니다. 파일 편집, Git 관리, 테스트 자동화, MCP 서버 연동까지 터미널 하나로 모든 개발 작업을 수행합니다.

✅ 장점
  • 가장 강력한 에이전트 기능
  • MCP로 외부 서비스 무제한 연동
  • 서브에이전트 병렬 작업
  • Git/GitHub 완벽 통합
  • CI/CD 파이프라인 통합
⚠️ 단점
  • 개발자 전용 (코딩 필수)
  • 터미널 사용 경험 필요
  • GUI 없음 (CLI만)
  • 유료 구독 또는 API 키 필요
⚙️ 개발자편

창업자·소상공인 💗 러버블 웹사이트·랜딩페이지를 직접 만들고 배포

4. 조합 활용

도구를 조합하면 더 효과적으로 작업할 수 있습니다.

조합 활용 방법 추천 대상
🧪 AI Studio → 💗 러버블 AI Studio로 빠르게 프로토타입 → 러버블로 풀스택 웹앱 완성 입문자
💗 러버블 → 💻 커서 러버블로 기본 구조 생성 → GitHub 연동 → 커서로 코드 커스터마이징 중급자
💻 커서 + ⚡ Claude Code 커서로 코딩 + Claude Code로 Git·테스트·배포 자동화 개발자
🚀 안티그래비티 → 💻 커서 에이전트로 빠르게 생성 → 커서로 세밀한 코드 제어 중급~개발자
💗 러버블 + 🚀 안티그래비티 러버블로 웹앱 + 안티그래비티로 모바일 앱 다중 플랫폼
💡 팁: 처음이라면 AI Studio → 러버블 순서로 시작하세요. AI Studio에서 바이브코딩 감각을 익히고, 러버블에서 본격적인 프로젝트를 만들 수 있습니다.
