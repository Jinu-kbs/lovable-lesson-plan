## 1. 바이브코딩이란?

**바이브코딩(Vibe Coding)**은 AI에게 자연어로 지시하여 소프트웨어를 만드는 새로운 개발 방식입니다. 코드를 직접 작성하지 않고, "이런 것을 만들어 줘"라고 말하면 AI가 코드를 생성합니다.

### 왜 주목받고 있을까?

| 기존 개발 방식 | 바이브코딩 |
|---------------|-----------|
| 프로그래밍 언어 학습 필요 | 자연어(한국어)로 지시 |
| 수백 줄의 코드를 직접 작성 | AI가 코드를 자동 생성 |
| 디버깅에 많은 시간 소요 | AI가 오류를 자동 수정 |
| 개발자만 가능 | 누구나 시작 가능 |

### 5대 바이브코딩 도구 비교

| 항목 | Lovable | AntiGravity | Cursor | AI Studio | **Claude Code** |
|------|---------|-------------|--------|-----------|----------------|
| **유형** | 웹 빌더 (노코드) | 에이전트 IDE | AI IDE | 웹 IDE | **에이전트 CLI** |
| **인터페이스** | 웹 브라우저 | 데스크톱 앱 | 데스크톱 앱 | 웹 브라우저 | **터미널** |
| **설치** | 불필요 | 데스크톱 설치 | 데스크톱 설치 | 불필요 | **npm 설치** |
| **코딩 필요** | 불필요 | 선택적 | 선택적 | 불필요 | **이해하면 유리** |
| **비용** | 무료~$20/월 | 무료 (프리뷰) | $20/월~ | 무료~ | **$20~$200/월** |

### 바이브코딩 학습 경로

```
AI Studio (입문) → Lovable (웹앱) → AntiGravity (에이전트) → Cursor (AI IDE) → Claude Code (CLI)
                                                                                  ↑ 지금 여기!
```

> **활용 가이드**: "바이브코딩은 코딩 실력이 없어도 아이디어만 있으면 시작할 수 있습니다. Claude Code는 터미널에서 AI와 대화하며 만드는 도구이지만, 초보자도 충분히 따라할 수 있습니다." — 활용 가이드

## 2. Claude Code란?

### 터미널에서 AI와 대화하는 코딩 도구

**Claude Code**는 Anthropic이 만든 **터미널 기반 AI 코딩 도구**입니다. 마치 AI 프로그래머와 터미널에서 채팅하듯이 프로젝트를 만들 수 있어요.

"터미널"이 낯설다고요? 걱정 마세요! 터미널은 컴퓨터에게 글자로 명령을 내리는 창입니다. 마우스로 클릭하는 대신, 글을 써서 컴퓨터와 대화하는 것이라 생각하면 됩니다.

```
나: "포모도로 타이머 웹앱을 만들어줘"
Claude Code: (파일을 만들고, 코드를 작성하고, 실행까지 해줍니다)
나: "타이머 색상을 파란색으로 바꿔줘"
Claude Code: (해당 파일을 찾아서 수정합니다)
```

### GUI vs CLI — 뭐가 다른가요?

| 항목 | GUI (러버블, 커서) | CLI (Claude Code) |
|------|-------------------|-------------------|
| **의미** | 마우스로 클릭하는 화면 | 글자를 입력하는 화면 |
| **비유** | 터치스크린 스마트폰 | 문자 메시지 |
| **장점** | 직관적, 쉬움 | 빠르고, 자동화 가능 |
| **단점** | 기능이 메뉴에 제한됨 | 명령어를 알아야 함 |

### Claude Code의 핵심 기능 (쉽게 설명)

| 기능 | 쉬운 설명 |
|------|----------|
| **AI 대화** | 터미널에서 한국어로 "이것 만들어줘" 하면 AI가 만들어줌 |
| **파일 생성/수정** | AI가 알아서 파일을 만들고, 코드를 작성하고, 수정함 |
| **명령어 실행** | AI가 필요한 프로그램 설치, 서버 실행 등을 대신 해줌 |
| **오류 수정** | 에러가 나면 AI가 원인을 분석하고 고쳐줌 |
| **Git 연동** | 코드를 저장하고 인터넷에 공개하는 것도 AI가 도와줌 |

> **핵심 비유**: Claude Code는 **카카오톡 같은 채팅창에서 AI 프로그래머와 대화하며 코딩하는 것**입니다. 다만 그 채팅창이 "터미널"이라는 곳에 있을 뿐이에요.

### 다른 도구와 비교하면?

| 도구 | 비유 | 작업 방식 |
|------|------|----------|
| **Lovable** | 인테리어 업체에 맡기기 | "이렇게 해주세요" → 완성품 받기 |
| **Cursor** | AI 조수와 함께 작업하기 | 에디터에서 코드를 보면서 AI가 도와줌 |
| **Claude Code** | AI 프로그래머와 채팅하기 | 터미널에서 대화하며 함께 만들기 |

## 3. 용어 해설

Claude Code를 사용하면서 만나는 용어들입니다. 어렵게 느껴지더라도 하나씩 익혀가면 됩니다!

| 용어 | 뜻 | 쉬운 비유 |
|------|-----|-----------|
| **CLI** | Command Line Interface. 글자로 명령하는 화면 | 문자 메시지로 컴퓨터에게 일 시키기 |
| **터미널** | CLI를 사용하는 프로그램 (PowerShell, Terminal) | 컴퓨터와 문자하는 앱 |
| **npm** | Node.js의 패키지 관리자 | 앱 스토어 (프로그램 설치용) |
| **Node.js** | JavaScript 실행 환경 | Claude Code가 돌아가는 엔진 |
| **Git** | 코드 변경 이력을 관리하는 도구 | 문서의 "변경 이력 추적" 기능 |
| **커밋(Commit)** | 변경 사항을 저장하는 행위 | 게임의 세이브 |
| **MCP** | AI에 외부 서비스를 연결하는 규격 | USB 포트 같은 연결 규격 |
| **API 키** | AI 서비스 이용 인증 번호 | 도서관 회원증 |
| **에이전트** | 스스로 판단하고 행동하는 AI | 일을 시키면 알아서 해결하는 직원 |
| **프롬프트** | AI에게 보내는 지시/요청 | AI에게 보내는 메시지 |
| **토큰** | AI가 텍스트를 처리하는 단위 | 글자 수 같은 것 (비용 계산용) |
| **CLAUDE.md** | Claude Code 설정 파일 | AI에게 주는 업무 매뉴얼 |

## 4. 사전 준비

### 필요한 것

| 항목 | 설명 | 필수 여부 |
|------|------|----------|
| **Node.js** | Claude Code가 실행되는 엔진 | 필수 |
| **Anthropic 계정 또는 API 키** | AI 서비스 인증 | 필수 |
| **Git** | 코드 저장 및 공유용 | 권장 |
| **텍스트 에디터** | 결과 확인용 (메모장도 OK) | 선택 |

### Step 1: Node.js 설치하기

Node.js는 Claude Code가 돌아가는 엔진입니다. npm(앱 스토어 같은 것)도 함께 설치됩니다.

1. [nodejs.org](https://nodejs.org) 접속
2. **LTS** 버전의 "Download" 버튼 클릭 (짝수 버전, 예: 20.x)
3. 다운로드된 설치 파일 실행 (Windows: `.msi`, Mac: `.pkg`)
4. 설치 마법사를 따라 완료
5. 터미널을 열고 확인:

```bash
node -v    # 예: v20.11.0 — 버전 번호가 나오면 성공!
npm -v     # 예: 10.2.4
```

> **터미널 여는 법**: Windows는 시작 메뉴에서 "PowerShell" 검색, Mac은 Spotlight(Cmd+Space)에서 "Terminal" 검색

### Step 2: Claude Code 설치하기

터미널에 아래 명령어를 입력하세요:

```bash
npm install -g @anthropic-ai/claude-code
```

> **"npm"이 뭔가요?** Node.js와 함께 설치되는 프로그램 설치 도구입니다. "앱 스토어에서 Claude Code를 설치해줘"라고 이해하면 됩니다.

설치 확인:

```bash
claude --version    # 버전 번호가 나오면 설치 성공!
```

### Step 3: 인증 설정하기

Claude Code를 처음 실행하면 인증이 필요합니다:

```bash
claude
```

| 인증 방법 | 설명 | 추천 대상 |
|----------|------|----------|
| **Claude.ai 로그인** | 브라우저에서 Anthropic 계정으로 로그인 | Max 구독자 |
| **API 키 입력** | API 키를 직접 입력 | 종량제 사용자 |

**방법 A — Claude.ai 로그인 (구독자용):**
1. "Login with Claude.ai" 선택 → 브라우저 자동 열림
2. Anthropic 계정으로 로그인 → 터미널로 돌아오면 인증 완료!

**방법 B — API 키 입력:**
1. [console.anthropic.com](https://console.anthropic.com)에서 API 키 발급
2. 터미널에서 환경변수 설정:

```bash
# Windows (PowerShell)
$env:ANTHROPIC_API_KEY="sk-ant-여기에-키를-붙여넣기"

# Mac/Linux
export ANTHROPIC_API_KEY="sk-ant-여기에-키를-붙여넣기"
```

### ✅ 사전 준비 체크리스트

- [ ] Node.js 설치 완료 (`node -v`로 확인)
- [ ] Claude Code 설치 완료 (`claude --version`으로 확인)
- [ ] 인증 설정 완료 (Claude.ai 로그인 또는 API 키)
- [ ] 터미널에서 `claude` 입력 시 환영 메시지 확인

## 5. 아이디어 구상하기

Claude Code는 터미널에서 AI와 대화하며 거의 모든 종류의 프로젝트를 만들 수 있습니다. 처음이라면 간단한 것부터 시작해봅시다!

### 직군별 추천 프로젝트

| 직군 | 추천 프로젝트 | 난이도 |
|------|-------------|--------|
| 교사 | 학급 안내 웹페이지, 시간표 생성기 | ★☆☆ |
| 디자이너 | 포트폴리오 페이지, 색상 팔레트 도구 | ★★☆ |
| 기획자 | 프로젝트 대시보드, 체크리스트 앱 | ★★☆ |
| 학생 | 자기소개 사이트, 스터디 타이머 | ★☆☆ |
| 사무직 | 업무 메모장, 간단한 계산기 | ★☆☆ |

### 프롬프트 예시 (프로젝트별)

**자기소개 웹페이지:**
```
자기소개 웹페이지를 만들어줘.
- 이름, 직업, 취미를 소개하는 깔끔한 페이지
- 프로필 사진 영역과 간단한 자기소개 문구
- 연락처 섹션 (이메일, SNS 링크)
- 모바일에서도 잘 보이게 반응형으로
- HTML, CSS, JavaScript만 사용
```

**할 일 관리 앱:**
```
간단한 할 일 관리 웹앱을 만들어줘.
- 할 일을 추가하고 삭제할 수 있는 기능
- 완료 체크 표시 기능
- 데이터는 브라우저에 저장 (localStorage)
- 심플하고 깔끔한 디자인
- HTML, CSS, JavaScript로 구현
```

**파일 정리 스크립트:**
```
다운로드 폴더를 자동으로 정리하는 Node.js 스크립트를 만들어줘.
- 파일 확장자별로 폴더 분류 (이미지, 문서, 영상 등)
- 실행하면 자동으로 파일을 해당 폴더로 이동
```

## 6. 프롬프트 작성하기

### 터미널에서 AI와 대화하는 법

Claude Code에서는 터미널 창에 한국어로 자연스럽게 말하면 됩니다. 카카오톡에서 채팅하듯이요!

```bash
$ claude

╭──────────────────────────────────────────╮
│ ✻ Welcome to Claude Code!                │
│   /help for available commands           │
╰──────────────────────────────────────────╯

> 안녕! 나는 코딩 초보인데, 간단한 웹사이트를 만들고 싶어.
```

### 알아두면 좋은 슬래시 명령어

터미널에서 `/`로 시작하는 특별한 명령어가 있어요:

| 명령어 | 기능 | 언제 쓰나요? |
|--------|------|-------------|
| `/help` | 사용 가능한 명령어 목록 보기 | 뭘 할 수 있는지 궁금할 때 |
| `/clear` | 대화 내용 초기화 | 새로운 주제로 대화하고 싶을 때 |
| `/compact` | 대화 내용 요약 | 대화가 너무 길어졌을 때 |
| `/cost` | 현재까지 사용한 비용 확인 | 비용이 궁금할 때 |

> **팁**: `/help`만 기억하세요! 나머지는 `/help`를 입력하면 전부 나옵니다.

### 한 문단으로 프로젝트 설명하기 (간단 PRD)

PRD(Product Requirements Document)는 "무엇을 만들지" 정리한 문서입니다. 아래 틀에 맞춰 한 문단으로 적어보세요:

```
[무엇을] 만들어줘.
- [핵심 기능 1]
- [핵심 기능 2]
- [핵심 기능 3]
- [디자인 요청]
- [사용할 기술] (잘 모르면 생략 OK!)
```

### 효과적인 프롬프트 패턴

**나쁜 예시:** `앱 만들어줘` (너무 모호해요!)

**좋은 예시:**
```
포모도로 타이머 웹앱을 만들어줘.
- 25분 집중 / 5분 휴식 타이머
- 시작, 일시정지, 리셋 버튼
- 완료한 세션 수 표시
- 깔끔하고 예쁜 디자인
- 순수 HTML/CSS/JavaScript로 만들어줘
```

### 대화를 이어가며 수정하기

Claude Code의 장점은 대화를 이어가며 계속 수정할 수 있다는 것입니다:

```
> 포모도로 타이머를 만들어줘 (첫 번째 요청)
(Claude가 만들어줌)

> 타이머 배경색을 다크 모드로 바꿔줘 (수정 요청)
(Claude가 수정함)

> 타이머가 끝나면 알림 소리가 나게 해줘 (기능 추가)
(Claude가 추가함)
```

> **활용 가이드**: "처음부터 완벽하게 요청하려고 하지 마세요. 대화를 이어가며 조금씩 수정하는 게 Claude Code를 잘 쓰는 방법입니다." — 활용 가이드

## 7. Claude Code 실습: 터미널에서 첫 프로젝트 만들기

이제 실제로 Claude Code를 사용해서 간단한 웹사이트를 만들어봅시다!

### Step 1: 터미널 열기

- **Windows**: 시작 메뉴에서 "PowerShell" 검색 → 클릭
- **Mac**: Spotlight (Cmd + Space)에서 "Terminal" 검색 → 클릭

> 글자를 입력할 수 있는 창이 뜨면 성공!

### Step 2: 프로젝트 폴더 만들고 이동하기

```bash
mkdir ~/Desktop/my-first-site    # 바탕화면에 폴더 만들기
cd ~/Desktop/my-first-site       # 그 폴더로 이동하기
```

> `mkdir` = 폴더 만들기(make directory), `cd` = 폴더 이동(change directory)

### Step 3: Claude Code 시작하기

```bash
claude
```

아래와 같은 환영 메시지가 나타나면 성공입니다:

```
╭──────────────────────────────────────────╮
│ ✻ Welcome to Claude Code!                │
│   /help for available commands           │
╰──────────────────────────────────────────╯

>
```

### Step 4: AI와 대화하며 프로젝트 만들기

Claude에게 웹사이트를 만들어달라고 요청해봅시다:

```
> 간단한 자기소개 웹페이지를 만들어줘.
> - 이름: 김바이브 (수정 가능)
> - 직업: 바이브코딩 입문자
> - 취미: AI 도구 탐험, 독서
> - 깔끔한 디자인, 배경은 연한 파스텔 톤
> - HTML, CSS, JavaScript 파일로 만들어줘
```

Claude가 작업을 시작하면 이런 메시지가 나옵니다:

```
Claude: index.html 파일을 생성하겠습니다.
[파일 생성 내용 표시]
이 변경을 허용할까요? (y/n)
```

> **중요**: Claude가 파일을 만들거나 수정할 때 `y` 또는 `n`으로 허용 여부를 묻습니다. 내용을 확인하고 `y`를 눌러주세요!

### Step 5: 결과 확인 및 수정

파일이 만들어졌으면 바탕화면의 `my-first-site` 폴더에서 `index.html`을 더블클릭하세요. 웹 브라우저에서 내가 만든 첫 웹사이트가 나타납니다!

마음에 안 드는 부분은 계속 대화하며 수정할 수 있어요:

```
> 배경색을 좀 더 진한 파란색으로 바꿔줘
> 이름 아래에 "바이브코딩으로 세상을 바꾸자!"라는 슬로건을 추가해줘
> 프로필 사진 자리에 동그란 원형 placeholder를 넣어줘
```

수정 후 브라우저에서 새로고침(F5)하면 변경된 결과를 확인할 수 있습니다.

> **활용 가이드**: "Claude Code로 처음 뭔가를 만들면 정말 신기해요! 터미널에서 말만 했는데 진짜 웹사이트가 나타나거든요." — 실사용자 후기

## 8. 프로젝트 공개하기 (배포)

만든 프로젝트를 인터넷에 공개하는 방법을 알아봅시다.

### 방법 1: GitHub Pages (초보자 추천)

Claude Code에게 직접 도움을 요청할 수 있습니다:

```
> 이 프로젝트를 GitHub에 올리고 GitHub Pages로 배포해줘.
> 저장소 이름은 'my-first-site'로 해줘.
```

Claude가 Git 초기화, 파일 추가, GitHub 저장소 생성, Pages 설정까지 단계별로 안내해줍니다.

> **참고**: GitHub 계정이 필요합니다. [github.com](https://github.com)에서 무료로 가입할 수 있어요.

### 방법 2: Vercel 배포 (간편)

```
> 이 프로젝트를 Vercel로 배포해줘
```

[vercel.com](https://vercel.com)에서 GitHub 계정으로 로그인하면 Claude 안내대로 자동 배포됩니다.

### 방법 3: Netlify 드래그 앤 드롭 (가장 쉬움)

[app.netlify.com/drop](https://app.netlify.com/drop)에 접속해서 프로젝트 폴더를 브라우저 화면으로 드래그 앤 드롭하면 자동으로 URL이 생성됩니다!

배포 완료 후 받은 URL(예: `https://my-first-site.vercel.app`)을 친구에게 보내서 자랑해보세요!

## 10. 문제가 생겼을 때 (트러블슈팅)

### 상황 1: 설치가 안 될 때 (Node.js 버전, 권한)

| 문제 | 해결 방법 |
|------|----------|
| `node`를 찾을 수 없다고 나옴 | Node.js 미설치 → [nodejs.org](https://nodejs.org)에서 LTS 설치 |
| Node.js 버전이 낮다고 나옴 | 18 이상 필요 → 최신 LTS 버전으로 재설치 |
| 권한 오류 (Permission denied) | Windows: 관리자 권한으로 PowerShell 실행 / Mac: `sudo npm install -g @anthropic-ai/claude-code` |

### 상황 2: 인증 에러 (API 키, 구독 확인)

| 문제 | 해결 방법 |
|------|----------|
| API 키 오류 | console.anthropic.com에서 키를 다시 복사하여 재설정 |
| 로그인 안 됨 | 브라우저에서 claude.ai에 먼저 로그인 확인 |
| 구독 필요 메시지 | Claude Pro/Max 구독 또는 API 키 필요 |

### 상황 3: 'claude' 명령을 찾을 수 없다고 할 때

```bash
# 방법 1: npx로 직접 실행 (설치 없이)
npx @anthropic-ai/claude-code

# 방법 2: 전역 설치 재시도
npm install -g @anthropic-ai/claude-code

# 방법 3: PATH 확인
npm config get prefix    # 이 경로가 시스템 PATH에 포함되어야 함
```

> **마지막 수단**: `npx @anthropic-ai/claude-code`로 실행한 뒤 "claude 명령어가 PATH에 등록이 안 돼. 도와줘"라고 하면 Claude가 해결 방법을 안내해줍니다.

## 11. 자주 묻는 질문 (FAQ)

### Q1. Claude Code는 무료인가요?

Claude Code 자체는 무료로 설치할 수 있지만, **사용하려면 Claude 구독 또는 API 키가 필요**합니다. Claude Pro($20/월)부터 사용 가능하며, API 키는 사용한 만큼만 지불합니다.

### Q2. 프로그래밍을 몰라도 쓸 수 있나요?

**네, 가능합니다!** 한국어로 "이것 만들어줘"라고 말하면 Claude가 코드를 작성합니다. 다만 기본적인 터미널 사용법(폴더 이동, 파일 확인)은 알면 훨씬 편합니다. 이 교안에서 필요한 것은 모두 설명하니 따라해보세요.

### Q3. 어떤 프로그래밍 언어를 지원하나요?

JavaScript, TypeScript, Python, Java, Go, Rust, C++ 등 **거의 모든 프로그래밍 언어를 지원**합니다.

### Q4. Cursor와 뭐가 다른가요?

| 항목 | Claude Code | Cursor |
|------|------------|--------|
| **화면** | 터미널 (글자 입력) | 에디터 (마우스 + 글자) |
| **비유** | AI와 문자로 대화 | AI와 함께 문서 편집 |
| **장점** | 자동화, 빠른 작업 | 시각적, 직관적 |
| **추천** | 터미널에 익숙한 분 | GUI를 선호하는 분 |

> 처음이라면 Cursor가 더 쉬울 수 있어요. 하지만 Claude Code는 익숙해지면 더 빠르고 강력합니다!

### Q5. 만든 프로젝트는 어디에 저장되나요?

**내 컴퓨터에 저장됩니다.** Claude Code를 시작한 폴더에 파일이 생성되며, 인터넷이 끊겨도 파일은 그대로 남아있습니다.

### Q6. 인터넷이 필요한가요?

**네, AI와의 대화에는 인터넷이 필요합니다.** 단, 만들어진 파일은 내 컴퓨터에 저장되므로 나중에 인터넷 없이도 볼 수 있습니다.

### Q7. 한국어로 대화할 수 있나요?

**네!** 일상적인 한국어로 지시하면 Claude가 이해하고 작업합니다.

### Q8. API 비용이 얼마나 드나요?

간단한 프로젝트 하나에 보통 **$0.5~$3** 정도입니다. `/cost` 명령어로 실시간 확인할 수 있으니 수시로 체크하세요.

## 12. 주의사항 및 팁

### 보안 관련 이해하기

Claude Code가 파일을 만들거나 수정할 때는 항상 **허용 여부를 물어봅니다**. 반드시 내용을 확인하고 허용해주세요!

| 항목 | 설명 |
|------|------|
| **승인 시스템** | Claude가 파일을 변경할 때마다 y/n으로 확인 요청 |
| **민감 정보 주의** | 비밀번호, API 키 등을 프롬프트에 직접 입력하지 마세요 |
| **코드 전송** | AI에게 보내는 내용은 Anthropic 서버로 전송됩니다 |

### 비용 관리

| 팁 | 설명 |
|----|------|
| `/cost` 자주 확인 | 현재 세션에서 사용한 비용을 확인하세요 |
| `/compact` 활용 | 대화가 길어지면 요약해서 토큰을 절약하세요 |
| 구체적으로 요청 | 모호한 요청은 왕복 대화가 늘어나 비용이 증가합니다 |
| 작은 단위로 작업 | 한 번에 큰 프로젝트를 요청하지 마세요 |

### 초보자가 자주 하는 실수

| 실수 | 올바른 방법 |
|------|-----------|
| 프롬프트가 너무 모호함 ("앱 만들어줘") | 기능, 디자인, 기술을 구체적으로 설명 |
| 폴더를 만들지 않고 시작 | 반드시 새 폴더를 만들고 그 안에서 시작 |
| 승인 없이 무조건 y 누름 | 어떤 파일을 변경하는지 한 번씩 확인 |
| 에러를 직접 검색 | Claude에게 에러 메시지를 보여주고 해결 요청 |
| 대화가 너무 길어짐 | `/compact`로 요약하거나 `/clear`로 새로 시작 |

### 실전 노하우

1. **작은 것부터 시작하세요**: 처음에는 간단한 웹페이지부터 만들어보세요
2. **대화를 이어가세요**: 한 번에 완벽하게 만들려 하지 말고, 조금씩 수정해나가세요
3. **에러를 두려워하지 마세요**: 에러가 나면 Claude에게 "에러가 났어. 고쳐줘"라고 하면 됩니다
4. **`/cost` 습관**: 비용이 걱정되면 수시로 `/cost`를 입력해서 확인하세요
5. **폴더 정리**: 프로젝트마다 새 폴더를 만들어서 깔끔하게 관리하세요
